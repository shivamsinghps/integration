h3 => [ 5) What do you understand by bias, variance trade-off? ]
p => [  When we work with a supervised machine learning algorithm, the model learns from the training data. The model always tries to best estimate the mapping function between the output variable(Y) and the input variable(X). The estimation for target function may generate the prediction error, which can be divided mainly into  Bias error , and  Variance error . These errors can be explained as: ]
strong => [ Bias Error: ]
strong => [ High Bias: ]
strong => [ underfitting ]
strong => [ Low Bias: ]
strong => [ low bias ]
strong => [ Variance Error: ]
strong => [ an error caused by the model's sensitivity to small fluctuation in training dataset ]
strong => [ Bias Variance tradeoff: ]
p => [ In the machine learning model, we always try to have low bias and low variance, and  ]
li => [ If we try to increase the bias, the variance decreases ]
li => [ If we try to increase the variance, the bias decreases. ]
p => [  Hence, trying to get an optimal bias and variance is called  bias-variance trade-off . We can define it using the Bull eye diagram given below. There are four cases of bias and variances: ]
li => [ If there is low bias and low variance, the predicted output is mostly close to the desired output. ]
li => [ If there is low bias and high variance, the model is not consistent. ]
li => [ If there is high variance and low bias, the model is consistent but predicted results are far away from the actual output. ]
li => [ If there is high bias and high variance, then the model is inconsistent, and also predictions are much different with actual value. It is the worst case of bias and variance.  ]
