h3 => [ 1) Explain Joomla. ]
p => [ Joomla is an open source Content Management System (CMS) written in PHP. It is a very popular CMS along with WordPress and Drupal. It facilitates users to develop online applications and websites. Joomla is built on model-view-controller (MVC) web application framework. ]
h3 => [ 2) Explain Mambots in Joomla.  ]
p => [ A Mambot is a small, task-oriented function, mini program or plugin. It is used to modify content before display, extend site search, add core functionality etc. Mambots are renamed plugins in Joomla 1.5. ]
h3 => [ 3) Explain categories of mambots.  ]

li => [ search ]

li => [ Joomla is open source and absolutely free to use.  ]
li => [ It provides a lot of free components are available in extensions.joomla.org/ ]
li => [ It has inbuilt permission and user role. ]
li => [ One click updation.  ]
li => [ Provide simple, elegant and user-friendly templates free to use.  ]
h3 => [ 5) Explain drawbacks of Joomla. ]
li => [ Provide limited options ]
li => [ Plugin compatibility may cause an issue. ]

li => [ Limited server resources and efficiency ]
