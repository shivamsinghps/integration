p => [ The version control system consists of a central shared repository where teammates can commit the changes to a file or set of files. It allows: ]
li => [ To revert the file to the previous state. ]
li => [ To compare changes over time.  ]
li => [ To revert the entire project to a previous state. ]
li => [ To see who modified something in the files. ]

h3 => [ 21) Explain pair programming regarding DevOps? ]

p => [  One programmer act as a  Driver  and other as an  Observer  who continuously monitors the progress of a project to identify problems. The roles can be reversed at any point of time without any prior intimation. ]

h3 => [ 23) Define DevOps Automation? ]
p => [ Automation is the crucial need for DevOps practices, and automate everything is the fundamental principle of DevOps. Automation kick starts from the code generation on the developer's machine, until the code is pushed to the code, to monitor the application and system in the production. ]
p => [ Automation in DevOps boosts speed, consistency, higher accuracy, reliability, and increases the number of deliveries. ]
