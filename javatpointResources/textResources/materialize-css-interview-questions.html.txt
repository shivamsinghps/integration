h3 => [ 1) What is Materialize CSS?  ]
p => [ Materialize CSS is a CSS framework which contains library files. It is created with CSS, JavaScript and HTML. It is used to create attractive, reliable, and functional webpages and web apps.  ]
h3 => [ 2) When was Materialize CSS released? ]
p => [ Materialized CSS was released on 8 November 2015.  ]
h3 => [ 3) What are the salient features of Materialize CSS? ]
p => [ Following is a list of some salient features of Materialize CSS:  ]
li => [ It contains In-built responsive designing. ]
li => [ It provides standard CSS with minimal footprint. ]
li => [ It provides new versions of common user interface controls such as buttons, checkboxes, and text fields adapted to follow Material Design concepts. ]
li => [ It facilitates enhanced and specialized features such as cards, tabs, navigation bars, toasts, and so on. ]
li => [ It is free to use and requires jQuery JavaScript library to function properly. ]
li => [ It is cross-browser supported, and can be used to create reusable web components. ]
h3 => [ 4) How do we use Materialize CSS? ]
p => [ There are two methods to use Materialize CSS:  ]
p => [  Local Installation:  It is achieved by downloading the materialize.min.css and materialize.min.js files on our local machine and include them in your HTML code. ]
p => [  CDN Based Version:  By including the materialize.min.css and materialize.min.js files into our HTML code directly from the Content Delivery Network (CDN). ]
h3 => [ 5) Which are the different Utility classes in Materialize CSS?  ]
p => [ A list of different Utility classes in Materialize CSS:  ]
p => [  Color utility classes:  Examples: .red, .green, .grey etc. ]
p => [  Alignment utility classes:  Examples: .valign-wrapper, .left-align, .right-align, .center-align, .left, .right etc. ]
p => [  Hiding Content utility classes as per device size:  Examples: .hide, .hide-on-small-only, .hide-on-med-only etc. ]
p => [  Formatting utility classes:  Examples: truncate, hoverable etc. ]
