h1 => [ Interview Tips ]
p => [ Job Interview is the most important thing for a job seeker. If you go for an interview you must be well prepared for this because you don't get a second chance to make a great first impression. Here we provide some strategies to improve your interview skills: ]
h2 => [ 1) Practice for good non-verbal communication ]
p => [ The non-verbal communication is used to demonstrate confidence. You should follow these instructions strictly: ]
li => [ Stand straight  ]
li => [ Make eye contact while answering ]
li => [ Make a firm handshake ]
li => [ Feel confident ]
h2 => [ 2) Dressing Sense ]
p => [ Power dressing is very important to impress the interviewer. It also improves your confidence level. You must be well-groomed. It is better for you to find out company dress code before going for interview. ]
li => [ Wear formal dresses ]
li => [ Black leather shoes (must be well-polished) ]
li => [ Have a professional, simple and nice hair cut ]
li => [ Cleanly Shaven ]
h2 => [ 3) Listen carefully ]
p => [ From the beginning of the interview you should listen to every word of the interviewer very carefully. If you are not listening properly, you are missing an opportunity to impress him because sometimes there is a hidden answer in the question already. So you should improve your listening power.  ]
h2 => [ 4) Don't talk too much ]
p => [ It is not a good practice to speak unnecessarily. Your answer must be to the point.  ]
li => [ Speak less but to the point ]
li => [ Don't ramble (You must be well prepared to avoid this situation) ]
li => [ Focus on your strong skills ]
li => [ Speak appropriate sentences ]
h2 => [ 5) Don't be too familiar ]
p => [ All interviewers like to talk in professional manner. You should not try to make a false familiar atmosphere. They are smart and catch all your cunningness in just a minute. So, don't try to be over smart and speak appropriately. ]
h2 => [ 6) Ask questions ]
p => [ Generally when the interviewers ask that if you have a question? Most of the candidates say no. It is a wrong practice. You should ask questions also according to the situation, it shows that you are listening properly and have a keen interest. Asking questions also gives you an opportunity to find the right place for you in the company. But always remember that it should not look artificial.  ]
h3 => [ Top 10 questions that can be asked by candidates ]
p => [ This is a list of top 10 interview questions which can be asked by the candidate when the interviewer done. Generally an interviewer gives a chance to the candidate to ask some questions at the end of the interview. The following questions are written for that situation. These questions make you able to get as much insight as possible into the position and the employer.  ]
li => [ Que: Can you please describe the responsibilities of the position? ]
li => [ Que: What are the biggest challenges of this job? ]
li => [ Que: Is this a new position? ]
li => [ Que: Can you please describe the company's management style? ]
li => [ Que: What are the prospects for the growth? ]
li => [ Que: When can I expect to hear from you? ]
li => [ Que: If I want to extend the job offer, how soon would you like me to start? ]
li => [ Que: What is the best part of working for this company? ]
li => [ Que: Is there any possibility for relocation? ]
li => [ Que: How much travel is expected? ]
p => [ Note: Although these are the possible questions to ask an interviewer but you should ask these questions according to the situation, experience and the job profile. Always remember that this is not mandatory to ask questions to interviewer. If you are fresher then it is better to shut your mouth or say only some words of appreciation for the interviewer.  ]
h3 => [ List of questions that should not to be asked ]
p => [ Following is a list of questions that you should avoid asking because they would not present you in a positive zone.  ]
li => [ Que: Did I get this job? ]
li => [ Que: Can I change my schedule if I get this job? ]
li => [ Que: What are the working areas of this company? ]
li => [ Que: If I get the job, can I take a vacation? ]
h2 => [ 7) Don't look desperate ]
p => [ Don't show an attitude that you are dying for this job. You should not look desperate and less confident. It is always beneficial for you to look cool, calm and confident within the whole session of interview. It also makes sure the interviewer that you have some ability and self-confidence and can do something.  ]
