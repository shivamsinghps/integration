p => [ The important properties of the XMLHttpRequest object are given below. ]
li => [ onReadyStateChange - It is called whenever readystate attribute changes. ]
li => [ readyState - It represents the state of the request. ]
li => [ responseText - It returns response as text. ]
li => [ responseXML - It returns response as XML. ]
li => [ status - It returns the status number of a request. ]
li => [ statusText - It returns the details of status. ]
a => [ More details. ]
h3 => [ 10) What are the important methods of XMLHttpRequest? ]
li => [ abort() - It is used to cancel the current request.  ]
li => [ getAllResponseHeaders() - It returns the header details.  ]
li => [ getResponseHeader() - It returns the specific header details.  ]
li => [ open() - It is used to open the request.  ]
li => [ send() - It is used to send the request. ]
li => [ setRequestHeader() - It adds request header. ]
a => [ More details. ]
h3 => [ 11) What are the types of open() method used for XMLHttpRequest? ]
li => [ open(method, URL) - It opens the request specifying get or post method and URL.  ]
li => [ open(method, URL, async) - It is same as above but specifies asynchronous or not.  ]
li => [ open(method, URL, async, username, password) - It is same as above but specifies the username and password.  ]
a => [ More details. ]

li => [ send() - It sends get request  ]
li => [ send(string) - It sends post request.  ]
a => [ More details. ]
h3 => [ 13) What is the role of the callback function in AJAX? ]
p => [ The callback function passes a function as a parameter to another function. If we have to perform various AJAX tasks on a website, then we can create one function for executing XMLHttpRequest and a callback function to execute each AJAX task. ]
h3 => [ 14) What is JSON in AJAX? ]
p => [ JSON stands for JavaScript Object Notation. In AJAX, it is used to exchange data between a browser and a server. It is easy to understand, and data exchange is faster than XML. It supports array, object, string, number, and values. ]
