h1 => [ Shell Scripting Interview Questions ]
p => [  A list of top frequently asked  Shell Scripting interview questions  and answers are given below.  ]
h3 => [ 1) What is shell script? ]
p => [ A shell script is a command containing text file that contains one or more commands.  ]
h3 => [ 2) Why is the use of shell script? ]
p => [ It is used by system administrators to issue numbers of commands to accomplish the task. All commands are added together in a text file (shell script) to complete daily routine tasks.  ]
h3 => [ 3) What are the advantages of shell scripting? ]
p => [ These are the two main advantages of shell scripting: ]
li => [ It facilitates you to develop your own operating system with relevant features best suited for you. ]
li => [ You can design software applications according to their platform.  ]
h3 => [ 4) What are the disadvantages of shell scripting? ]
p => [ Following are the main disadvantages of shell scripting: ]
li => [ A weak design can destroy the entire process and could prove a costly error. ]
li => [ If a typing error occurs during the creation then it can delete the entire data as well as partition data.  ]
li => [ Its initial process is slow and improves gradually.  ]
li => [ Portability between different operating system is difficult.  ]
h3 => [ 5) What are the different types of variables used in shell script? ]
p => [ There are two types of variables used in shell script: ]
li => [ System defined variables: These variables are defined or created by operating system itself.  ]
li => [ User defined variables: These variables are defined by system users. ]
h3 => [ 6) What is the syntax of "nested if statement" in shell scripting? ]
h3 => [ 7) What is the use of "$?" sign in shell script? ]
p => [ While writing the shell script, the "$?" sign is used with "if statement" to check whether the previous command is executed successfully or not.  ]
h3 => [ 8) What is the use of break command? ]
p => [ The break command is used to escape from a loop in progress.  ]
h3 => [ 9) What is GUI scripting? ]
p => [ GUI stands for Graphical User Interface. It is used for controlling a computer and its applications. GUI scripting supports different applications. It mostly depends on the operating system.  ]
h3 => [ 10) What are the various stages of a Linux process it passes through? ]
p => [ A Linux process generally passes through four stages: ]
strong => [ Waiting ]
strong => [ Running ]
strong => [ Stopped ]
strong => [ Zombie ]
h3 => [ 11) Is it possible to substitute "ls" command in the place of "echo" command? ]
p => [ Yes.  ]
h3 => [ 12) What is the syntax of while loop in shell scripting? ]
p => [ The while loop is used to repeat its block of commands a number of times. It is different from for loop, it iterates until the while condition is no longer true.  ]
h3 => [ 13) What is the difference between soft and hard links? ]
p => [  Inode:  Each file contains an index number known as "inode" which is comprised of metadata about the associated file. ]
p => [  Hard Link:  is the mirror of original linked file. In such case, hard link and original file contains same inode. Hard link remains linked whether original file is removed. ]
p => [  Soft Link:  is like shortcut feature of Windows OS. In such case, hard link and original file contains different inode. Soft link will not work correctly if original file is removed. ]
h3 => [ 14) What is the use of "$#" in Shell Scripting? ]
p => [ The $# shows total number of arguments passed to the script. ]
h3 => [ 15) What are the Zombie processes? ]
p => [ Zombie processes are the processes which are died but exit status is not picked by the parent process. ]
h3 => [ 16) What are the 3 standard streams in Linux? ]
p => [ 0 - Standard Input  ]
p => [ 1 - Standard Output  ]
p => [ 2 - Standard Error  ]
