h1 => [ Unix Interview Questions ]
p => [  A list of top frequently asked  Unix interview questions  and answers are given below.  ]
h3 => [ 1) What is Unix? ]
p => [ UNIX is a portable operating system that is designed for efficient multitasking and multi-user functions. Since it is a portable operating system, it can run on different hardware platforms. It is written in C language. It was developed by Ken Thompson, Dennis Ritchie, Douglas McIlroy, and Joe Ossanna. ]
h3 => [ 2) List the distributions of UNIX. ]
p => [ UNIX has many distributions including Solaris UNIX, AIX, HP UNIX and BSD and many more. ]
h3 => [ 3) List some features of UNIX. ]
p => [ UNIX includes the following features: ]
p => [  UNIX supports the multiuser system:  In UNIX it is possible that many users can use the system with their separate workspace and logins i.e.it has full support for the multiuser environment. ]
p => [  UNIX supports the multitasking environment:  In UNIX many apps can run at a single instance of time this is also known as a multitasking environment. ]
h3 => [ 4) What are the core concepts of UNIX ]
p => [ The core concepts of UNIX are given below.  ]
strong => [ Kernel- ]
strong => [ Shell- ]
strong => [ Commands and Utilities- ]
strong => [ Directories- ]
h3 => [ 5) What is a UNIX shell? ]
p => [ The UNIX shell is a program which is used as an interface between the user and the UNIX operating system. It is not a part of the kernel, but it can communicate directly with the server. ]
h3 => [ 6) What is filter? ]
p => [ A filter is a program that takes input from standard inputs and performs some operation on that input to produce a result as standard output.  ]
h3 => [ 7) What are the devices represented in UNIX? ]
p => [ All devices in UNIX are represented by particular files that are located in /dev directory. ]
h3 => [ 8) Is there any method to erase all files in the current directory, along with its all sub-directories, by using only one command? ]
p => [ Yes, you should use "rm-r*" command for this purpose.  ]
p => [ Here, the "rm" command is used for deleting files, the -r option will erase directories and subdirectories with their internal data and * is used for selecting all entries. ]
h3 => [ 9) What is necessary before running a shell script from the terminal? ]
p => [ You must make the shell script executable by using the UNIX "chmod" command. ]
h3 => [ 10) How to terminate a shell script if statement? ]
p => [ A shell script if statement can be terminated by using "fi." ]
h3 => [ 11) Write down some common shells with their indicators? ]
li => [ sh - Bourne shell ]
li => [ csh - C Shell ]
li => [ bash - Bourne Again Shell ]
li => [ tcsh - enhanced C Shell ]
li => [ zsh - Z Shell ]
li => [ ksh - Korn Shell ]
h3 => [ 12) What are the main features of Korn Shell? ]
li => [ Arrays ]
li => [ Job control ]
li => [ Command Aliasing  ]
li => [ String manipulation ability ]
li => [ Built-in integer arithmetic ]
h3 => [ 13) What is the difference between cat command and more command? ]
p => [ The cat command is used to display the file contents on the terminal, whereas more command is used like a pager which displays the screen page by page If the file is large and you have to scroll off the screen before you view it. ]
h3 => [ 14) Which command is used to restrict incoming messages? ]
p => [ The "mesg" command is used to restrict incoming messages.  ]
h3 => [ 15) Which command is used to kill the last background job? ]
p => [ The "kill $!" Command is used to kill the last background job. ]
h3 => [ 16) Which data structure is used to maintain the file identification? ]
p => [ The "inode" data structure is used to maintain the file identification. Each file has a separate inode and a unique inode number. ]
h3 => [ 17) What a pipe? ]
p => [ A pipe is used to join two or more commands by using a pipe "I" character. The output of the first command is propagated to the second command through the pipe.  ]
h3 => [ 18) What are the links and symbolic links in a UNIX file system? ]
p => [ A link is a second name for a file. Links are used to assign more than one name to a file, but cannot be used to designate a directory more than one name or link filenames on different computers. ]
p => [ Symbolic links are the files that only contain the name of another file. The operations on the symbolic link are directed to the file pointed by it. Both the limitations of connections are eliminated in symbolic links. ]
h3 => [ 19) Explain system bootup in UNIX. ]
p => [ System bootup is the first thing that takes place when the power button is pressed in UNIX. Whenever the power button is pressed, BIOS is fired up and checks if all the hardware connected to the system are working correctly, after being successful the system asks the user to provide authentication.  ]
h3 => [ 20) How to change the password in UNIX operating system? ]
p => [ To change the password in UNIX operating system : ]
li => [ Type in the command passwd. ]
li => [ You will get a screen which prompts to enter your default(current) password, type your current password. ]
li => [ if the current password is verified, then the terminal will prompt you to enter the new password. ]
li => [ Enter the new password twice, and your password will be updated. ]
h3 => [ 21) How to list directories in UNIX? ]
p => [ Command ls can be used to list directories in command prompt. Also, we can also use a variety of ls commands like: ]
td => [ ls -a ]
td => [ In Linux, hidden files start with. (dot) Symbol and they are not visible in the regular directory. The (ls -a) command will enlist the whole list of the current directory including the hidden files. ]
td => [ ls -l ]
td => [ It will show the list in a long list format. ]
td => [ ls -lh ]
td => [ This command will show you the file sizes in human readable format. Size of the file is tough to read when displayed regarding a byte. The (ls -lh)command will give you the data regarding Mb, Gb, Tb, etc. ]
td => [ ls -lhS ]
td => [ If you want to display your files in descending order (highest at the top) according to their size, then you can use (ls -lhS) command. ]
td => [ ls -l - -block-size=[SIZE] ]
td => [ It is used to display the files in a specific size format. Here, in [SIZE] you can assign size according to your requirement. ]
td => [ ls -d */ ]
td => [ It is used to display only sub directories. ]
td => [ ls -g or ls -lG ]
td => [ With this, you can exclude column of group information and owner. ]
td => [ ls -n ]
td => [ It is used to print group ID and owner ID instead of their names. ]
td => [ ls --color=[Value] ]
td => [ This command is used to print list as colored or discolored. ]
td => [ ls -li ]
td => [ This command prints the index number if the file in the first column. ]
td => [ ls -p ]
td => [ It is used to identify the directory easily by marking the directories with a slash (/) line sign. ]
td => [ ls -R ]
td => [ It will display the content of the sub-directories also. ]
td => [ ls -lX ]
td => [ It will group the files with the same extensions together in the list. ]
td => [ ls -lt ]
td => [ It will sort the list by displaying a recently modified file at the top. ]
td => [ ls ~ ]
td => [ It gives the contents of the home directory. ]
td => [ ls ../ ]
td => [ It gives the contents of the parent directory. ]
td => [ ls --version ]
td => [ It checks the version of ls command. ]
h3 => [ 22) How to check the date in UNIX? ]
p => [  To display the date in UNIX use the  date  command in command prompt. ]
h3 => [ 23) How to log out in UNIX? ]
p => [  To log out of UNIX type the  logout  command in the command prompt. ]
h3 => [ 24) How to perform a system shutdown in UNIX? ]
p => [ To perform system shutdown in UNIX, you can use the following commands: ]
li => [ halt ]
li => [ init 0 ]
li => [ init 6 ]
li => [ power off ]
li => [ reboot ]
li => [ shutdown ]
h3 => [ 25) How many types of files are there in UNIX? ]
p => [ There are three kinds of files in UNIX: ]
li => [ Ordinary files: An ordinary file is the one which contains data, text or program instructions. ]
li => [ Directories: These include both ordinary files and special files. ]
li => [ Special Files: These are the files which provide unique access to hardware such as hard drives, CD-Rom Drives e.t.c. ]
h3 => [ 26) What are hidden files in UNIX? ]
p => [ Hidden files in UNIX are the files which have a .(dot) before the file name. These files do not show up in the traditional file manager. ]
p => [ Common examples of hidden files are: ]
li => [ .profile ]
li => [ .kshrc ]
li => [ .rhosts ]
li => [ .cshrc ]
h3 => [ 27) What is the difference between a single dot and double dot in UNIX? ]
p => [  .(Single dot) -represents the current directory ]
p => [  ..(Double dot) -represents the parent directory. ]
h3 => [ 28) How to create files in UNIX? ]
p => [ Creating files in UNIX is simple. The User needs to use the vi editor to create new files. ]
p => [  Type  vi filename  in command prompt to create new files. We can also use the  touch  command to create a zero byte file.  ]
h3 => [ 29) How to display the contents of a file? ]
p => [ The user can use the cat command followed by the filename to display the command of a file. This command should be entered in the command prompt. The syntax of the command is shown below.  ]
strong => [ $ cat filename ]
p => [ Where the cat is the command to view contents of the file specified by the filename.
Also if you want the line number to be displayed along with the content, you can use cat command with option -b. ]
h3 => [ 30) How to calculate the number of words in a file? ]
p => [ To count the number of words in a file, Use the following command. ]
strong => [ $ wc filename ]
p => [  Where wc is the command to count the number of words in the file specified by  filename . ]
h3 => [ 31) How to create a blank file in UNIX? ]
p => [ Blank files can be created by using the touch command, the syntax for the touch command is as follows: ]
strong => [ $ touch filename ]
h3 => [ 32) How to know the present working directory in UNIX? ]
p => [ To know the present working directory, Run the following command on the terminal.  ]
strong => [ $ pwd  ]
h3 => [ 33) How to know the information about a file? ]
p => [ To fetch the information about a file, use the following command. ]
strong => [ $ file filename ]
h3 => [ 34) How to change the directory in UNIX? ]
p => [ To change the directory, you can use the cd command in the terminal window. It changes the current directory to the specified directory. ]
strong => [ $ cd directory-name ]
h3 => [ 35) How to move files from one directory to other in UNIX? ]
p => [ In UNIX, mv command is used to move the file from one directory to some other directory. ]
strong => [ $ mv &lt;file-name&gt; &lt;destination path&gt; ]
h3 => [ 36) How to copy files from one directory to other in UNIX? ]
p => [ In UNIX, cp command is used to copy a file from one directory to some other directory. The syntax of the cp command is given below. ]
strong => [ $ cp -r source filename destination file name.  ]
p => [ The -r is used to copy all the content of a directory including sub-directories recursively.  ]
h3 => [ 37) How to remove files in UNIX? ]
p => [ To remove files, you can use the rm command. The syntax of the rm command is given below. ]
strong => [ $ rm &lt;filename&gt;  ]
strong => [ we can use -r with the rm command to delete all the sub-directories recursively. ]
h3 => [ 38) How to make a new directory in UNIX? ]
p => [ To make a new directory, you can use the mkdir command.  ]
strong => [ $ mkdir &lt;directory-name&gt; ]
h3 => [ 39) How to remove the directory in UNIX? ]
p => [ To remove the directory, you can use the rmdir command. To use this command, use the following syntax.  ]
strong => [ $ rmdir filename. ]
