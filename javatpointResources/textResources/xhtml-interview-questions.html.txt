p => [ There are some changes in XHTML as compared to HTML: ]
li => [ All documents must have a DOCTYPE.  ]
li => [ The xmlns attribute in &lt;html&gt; is mandatory and must specify the xml namespace for the document. ]
li => [ &lt;html&gt;, &lt;head&gt;, &lt;title&gt;, and &lt;body&gt; are mandatory with their respective closing tags. ]

li => [ All XHTML attributes must be in lower case. ]
li => [ The name attribute has changed. ]
li => [ XHTML attributes cannot be shortened. ]
li => [ XHTML attribute values must be quoted. ]

p => [ Following are the reasons specifying why XHTML is better than HTML: ]

li => [ XML of XHTML makes easy the integration of new elements as subsets of SGML. ]
