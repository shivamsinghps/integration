h1 => [ Microsoft Access Interview Questions ]
p => [  A list of top frequently asked  Microsoft Access interview questions  and answers are given below. ]
h3 => [ 1) What is Microsoft Access database? ]
p => [ Microsoft Access database is a relational database management system which combines GUI (Graphical User Interface) with Microsoft Jet database engine. It can import and use data from Access, SQL, Oracle, etc.  ]
p => [ This software is used to build application software.  ]
h3 => [ 2) What is the way to link the explicit data into Access database?  ]
p => [ Follow the instructions given below to link explicit data into Access database: ]
li => [ Go to main menu and click on EXTERNAL DATA within Import and Link group. ]
li => [ Import the required file. ]
li => [ Follow the instruction in the Get External Data wizard to know about the file that can link Access database. ]
h3 => [ 3) What are the several file extensions in MS Access? ]
p => [ Following are the six file extensions in MS Access: ]
li => [ Access workgroup.mdw ]
li => [ Access database.accdb ]
li => [ Access Workgroup.mdw ]
li => [ Access blank project template.adn ]
li => [ Access project.adp ]
li => [ Protected access database.accde ]
h3 => [ 4) What is maximum size for an Access database? ]
p => [ Maximum size of Access 2.0 version database is 1 gigabyte. ]
h3 => [ 5) Which technique is used to increase the size of database? ]
p => [ Replication is used to increase the size of database.  ]
h3 => [ 6) Tell some of the most important uses of MS Access? ]
p => [ Uses of MS Access: ]
li => [ MS Access is used as frontend when the backend is product of ODBC compliant. ]
li => [ MS Access is widely used for small database applications are based on web and hosted on IIS by making use of ASP.NET pages. ]
li => [ Mainly, it is used by small firms and businesses, specific departments in large organizations and some people for developing applications on desktop for data applications. ]
li => [ It can be used as RAD while developing standalone applications and prototype. ]
h3 => [ 7) What is a super key? How is it different from a candidate key? ]
p => [ A superkey is a set of attributes within a table whose values can be used to uniquely identify a tuple. On the other hand, a candidate key is a minimal set of attributes mandatory to identify a tuple. ]
p => [ A candidate key is also called a minimal super key. ]
h3 => [ 8) What are the various type of relations between different data? ]
p => [ The data within the table can build only three types of relationships: ]
strong => [ One-to-one relationship: ]
strong => [ One-to-many relationship: ]
strong => [ Many-to-many relationship: ]
h3 => [ 9) What are the protection features of MS Access? ]
p => [ MS Access protection features are used to protect some documents by locking them. If you do not want to change some document then you can lock the document containing data and its code from modifications by changing file system extension to .MDE. Only the MDE extension files can be changed. One can find specific tools in the market that can be used to decompile and unlock code but changes made to VB scripts are permanent. ]
h3 => [ 10) How can you maintain a relationship between different databases in Access? ]
p => [ Whenever a relationship is established between different databases, the first table of one database considered as a primary key whereas the new field in another database considered as a foreign key. ]
h3 => [ 11) What are queries in MS Access? ]
p => [ Queries can be seen as a request used to perform various operations on data such as sort data, insert data, retrieve data, update data, delete data, etc. ]
h3 => [ 12) How to create a simple query in MS Access 2013? ]
p => [ Follow these instructions to create a simple query:  ]
li => [ Open database. ]
li => [ Click on -create tab. ]
li => [ Go to query wizard icon. ]
li => [ Select a query type. ]
li => [ Select the appropriate table from the pull-down menu. ]
li => [ Now, choose the required field in the query results. ]
li => [ You can insert information from additional tables, by repeating step 5 and 6. ]
li => [ Go to next. ]
li => [ Choose the result that you would like to produce. ]
li => [ Specify a title to your query. ]
li => [ Now, click finish. ]
h3 => [ 13) How can you export data in Excel format? ]
p => [ Follow the below steps to export data in Excel format:  ]
li => [ Click on toolbar and follow the External Data tab. ]
li => [ Now, go to Excel on the Export group. ]
li => [ After that excel spreadsheet dialog box will be opened on your screen. ]
li => [ Now, it is required to provide destination of the file name and format. ]
li => [ Click OK. Now data will be exported to Excel spreadsheet. ]
h3 => [ 14) How a table can be created using Microsoft Access Object Library (MAOL)?  ]
p => [ Define a variable of type object and initialize it with CreateTableDef() method. ]
h3 => [ 15) How can you create a form in MS Access 2013? ]
p => [ Follow these instructions to create a form: ]
li => [ Go to specific table you want to use to create a form.  ]
li => [ Click on Create tab and then locate the Forms group. ]
li => [ Click the Form Command after which a form will be created and opened in Layout view. ]
li => [ Click Save on the Quick Access toolbar and save the file with a specific name. ]
h3 => [ 16) What is the use of Pivot table? ]
p => [ Pivot table is used for improving the presentation of data. By using Pivot table, you can view data multi-dimensionally. A Pivot table can summarize the data by categories and display trends spread over a range of categories. ]
h3 => [ 17) What are the several methods to enter data in MS Access 2013? ]
p => [ Different methods to enter data in MS Access are: ]
li => [ Using SQL View ]
li => [ Using form ]
li => [ Datasheet View ]
h3 => [ 18) Would you easily automate your MS Access database through menu driven selections? ]
p => [ You can do it by creating a form with customized buttons that point to macros. An easier way is to use MS Access' built in function called "Switchboard Manager". This can be located in Tools/Add-Ins. ]
h3 => [ 19) What is the role of WHERE clause in MS Access? ]
li => [ WHERE clause is used with the SELECT statement to retrieve a value with certain criteria. ]
li => [ WHERE clause can be used to join different fields of dissimilar data types by using LIKE keywords where one field is treated as a criteria for another field. ]
h3 => [ 20) What are the advantages of MS Access over MS SQL Server? ]
p => [ Following are some of the advantages of MS Access over MS SQL Server: ]
li => [ MS Access is much easier to handle. ]
li => [ MS Access is more reliable as client can read and write directly into raw data table.  ]
li => [ MS Access is more scalable when number of users are less. ]
li => [ MS Access is cheaper and more compatible. ]
