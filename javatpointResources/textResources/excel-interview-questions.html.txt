h3 => [ 1) What is Microsoft Excel?  ]
p => [ Microsoft Excel is an electronic worksheet or spreadsheet application which is used for organizing, storing, and manipulating and analyzing data. It is developed by Microsoft.  ]
h3 => [ 2) What are cells? ]
p => [ The area where data is stored is known as cell.  ]
h3 => [ 3) Does each cell have unique address? ]
p => [ Yes, each cell has a unique address depends on the row and column value of the cell.  ]
h3 => [ 4) How can you add cells, rows or columns in Excel? ]
p => [ If you want to add a cell, row or column in Excel, right click the cell you want to add to and after that select insert from the cell menu. The insert menu makes you able to add a cell, a column or a row and to shift the cells affected by the additional cell right or down.  ]
h3 => [ 5) How would you format a cell? What are the options? ]
p => [ A cell can be formatted by using the format cells options. There are 6 format cells options: ]
li => [ Number ]
li => [ Alignment ]
li => [ Font ]
li => [ Border ]
li => [ Fill ]
li => [ Protection ]
h3 => [ 6) What is the use of comment? How to add comments to a cell? ]
p => [ Comments are used for a lot of reasons: ]
li => [ Comments are used to clarify the purpose of the cells. ]
li => [ Comments are used to clarify a formula used in the cell.  ]
li => [ Comments are used to leave notes for others users about a cell.  ]
p => [ To add a comment: Right click the cell and choose insert comment from the cell menu. Type your comment.  ]
h3 => [ 7) What does the red triangle indicate at the top right hand corner of the cell? ]
p => [ The red triangle at the top right hand corner of a cell indicates that there is a comment linked to the particular cell. If you put your cursor on it, it will show the comment.  ]
h3 => [ 8) How would you add comments to a cell? ]
p => [ To add a comment to a cell, you right click the cell and choose insert comment from the cell menu. Type your comment in the comment area provided. A red triangle at the top right hand corner of a cell indicates that there is a comment linked to that particular cell. To remove a comment from a cell, right lick the cell and then select delete comment from the cell menu. ]
h3 => [ 9) What are charts in MS Excel? ]
p => [ Charts are used to enable graphical representation of the data in Excel. A user can use any chart type, including column, bar, line, pie, scatter, etc. by selecting an option from Insert tab?s Chart group. ]
h3 => [ 10) What is Freeze Panes in MS-Excel? ]
p => [ Freeze Panes are used to lock any row or column. The locked row or column will be visible on the screen even after we scroll the sheet vertically or horizontally. ]
h3 => [ 11) Which are the different workbook protection types in Excel? ]
p => [ There are three ways to protect a workbook in Excel: ]
li => [ Password protection for opening a workbook ]
li => [ Protection for adding, deleting, hiding and unhiding sheets ]
li => [ Protection from changing size or position of windows. ]
h3 => [ 12) What is the difference among COUNT, COUNTA, COUNTIF and COUNTBLANK in Ms-Excel? ]
p => [  COUNT  is used to count cells containing numbers, dates, etc. any value stored as number excluding blanks. ]
p => [  COUNTA  or Count All is used to count any cell value containing numbers, text, logical values, etc. any type of value excluding blanks. ]
p => [  COUNTBLANK  count blank cells or cells with an empty string. ]
p => [  COUNTIF and COUNTIFS  count cells matching a certain criteria. ]
h3 => [ 13) What is Ribbon? ]
p => [ The ribbon specifies an area which runs along the top of the application and contains menu items and toolbars available in Excel. The ribbon has various tabs that contain groups of commands for use in the application.  ]
h3 => [ 14) Is it possible to hide or show the ribbon? ]
p => [ You can hide or show (minimize or maximize) the ribbon by pressing CNTRL F1. ]
h3 => [ 15) How to prevent someone from copying the cell from your worksheet? ]
p => [ If you want to protect your worksheet from being copied, go into Menu bar &gt; Review &gt; Protect sheet &gt; Password.  ]
p => [ By entering password you can prevent your worksheet from getting copied.  ]
h3 => [ 16) How to sum up the rows and column number quickly in the Excel sheet? ]
p => [ The SUM function is used to get the total sum of the rows and columns, in an excel worksheet. ]
h3 => [ 17) How can you resize the column? ]
p => [ There are two ways to resize a column: ]
li => [ To change the width of one column, drag the boundary on the right side of the column until you find your desirable width. ]
li => [ Select the Format from the home tab, and in Format, select the AUTOFIT COLUMN WIDTH under cell section. Click on this to change the cell size.  ]

h3 => [ 21) What does the IF function in Excel? ]
p => [ IF function is used in Excel to check whether certain conditions are true or false. If the condition is true then it will give the result accordingly and if the condition is false the result or output will be different.  ]

li => [ Press Ctrl+- on the keyboard to delete the blank columns. ]
