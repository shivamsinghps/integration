h1 => [ OpenStack Interview Questions ]
p => [  A list of top frequently asked  OpenStack interview questions  and answers are given below.  ]
h3 => [ 1) Explain OpenStack. ]
p => [ OpenStack is an open source and free set of software tools or cloud computing platform which is used for managing and building cloud computing platform for private and public cloud. ]
p => [ OpenStack is referred as the future of Cloud Computing. ]
h3 => [ 2) What are the modular architectural components of OpenStack? ]
p => [ Following is a list of OpenStack modular architectural components:  ]
li => [ Dashboard ]
li => [ Compute ]
li => [ Networking ]
li => [ Object Storage ]
li => [ Block Storage ]
li => [ Identity service  ]
li => [ Image Service ]
li => [ Telemetry ]
li => [ Orchestration ]
li => [ Database Service etc. ]
h3 => [ 3) What are the advantages/benefits of using OpenStack? ]
p => [ Advantages/Benefits of using OpenStack: ]
li => [ OpenStack can be used to develop any software as a service (SAAS) applications, for new developments or to improve existing solutions.  ]
li => [ It can be used as a strong foundation to deliver self-service storage to IT users. ]
li => [ It provides easy to handle storage at lower costs. ]
li => [ It can deliver on-demand objective or block storage with higher scalability. ]
li => [ An enterprise can save a lot of licensing fees by switching virtual machines running on VMware to OpenStack. ]
h3 => [ 4) What is "role" and "tenant" in OpenStack?  ]
p => [  role:  It specifies the authorization level of the user. ]
p => [  Tenant:  It specifies a group of users.  ]
h3 => [ 5) What are the storage types allowed by OpenStack compute? ]
p => [ OpenStack supports two types of storage: ]
p => [ 1.	Persistent Storage or volume storage ]
p => [ 2.	Ephemeral Storage  ]
p => [  Persistent Storage / Volume Storage:  It is persistent and independent of any particular instance. This storage is created by users. There are three types of persistent storage:  ]
strong => [ Object storage: ]
strong => [ Block storage: ]
strong => [ Shared File System storage: ]
p => [  Ephemeral Storage:  The ephemeral storage specifies a single instance. It is a temporary and short-lived storage that is disappeared once the VM is terminated. ]
h3 => [ 6) What is hypervisor? Which type of hypervisors are supported in OpenStack? ]
p => [ Hypervisor is a software or hardware tool which is used to create and run a virtual machine. OpenStack supports a variety of hypervisors like VMware, Citrix, and Microsoft etc. ]
h3 => [ 7) Which is the most important identity service in OpenStack? ]
p => [ Keystone is the most important and preferred Identity Service in OpenStack. It executes the complete OpenStack Identity API. ]
h3 => [ 8) What are the different networking options used in OpenStack? ]
p => [ The networking options used in Open Stack are: ]
strong => [ Flat DHCP Network Manager: ]
strong => [ Flat Network Manager: ]
strong => [ VLAN Network Manager: ]
h3 => [ 9) Which commands are used to pause and un-pause (resume) an instance? ]
p => [ For Pausing: $nova pause INSTANCE_NAME ]
p => [ For Un-pausing: $novaunpause INSTANCE_NAME ]

h3 => [ 11) What is Token in OpenStack? ]
p => [ Token is a type of authentication like password-based validation. It is generated when the user inserts the credential and authenticate as a keystone user then Tokens can be used to access OpenStack services without any revalidation.  ]
h3 => [ 12) How can you create a Token? ]
p => [  Users first need to authenticate their Keystone credentials to create a token.  ]
h3 => [ 13) Explain is OpenStack Python SDK? ]
p => [ Python SDK (Software Development Kit) is used to help users to write applications for performing automation tasks in Python by calling Python objects. ]
p => [ It provides a platform to work with multiple OpenStack services at one place. ]
h3 => [ 14) Explain is the role of API server in OpenStack? ]
p => [ In OpenStack, an API server provides an interface for the external world to interact with the cloud infrastructure. ]
h3 => [ 15) What are the commands used to generate key pairs in OpenStack? ]
p => [ Commands used to generate key pairs in OpenStack: ]
li => [ ssh-keygen ]
li => [ cd.ssh ]
li => [ nova keypair-add -pub_key id_rsa.pub mykey ]
h3 => [ 16) Which hardware is required for networking in OpenStack? ]
p => [ In OpenStack,networking can be done with following hardware: ]
li => [ Networks ]
li => [ Routers ]
li => [ Subnets ]
li => [ Ports ]
li => [ Vendor Plugins ]
h3 => [ 17) Which command is used to manage floating IP addresses in OpenStack? ]
p => [ nova floating-ip-* ]
h3 => [ 18) Explain the usage of Cinder in OpenStack? ]
p => [ OpenStack Cinder is used to handle block storage in the context of OpenStack. ]
h3 => [ 19) What is the use of $ nova floating-ip-pool-list command in OpenStack? ]
p => [ The $ nova floating-ip-pool-list command is used to list IP address information in OpenStack. ]
h3 => [ 20) Explain the term "flavor" in OpenStack? ]
p => [ The term "flavor" is an available hardware configuration for a server, which defines the size of a virtual server that can be launched. ]
