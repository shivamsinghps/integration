h3 => [ 15) How to create a list with a column in SharePoint? ]
p => [ There are the following steps to create a list in SharePoint ? ]
p => [  Step 1:  Go to the  Home page  of your site, click on the  +New,  a drop-menu will appear, in which select  List  option. ]
p => [  Step 2:  Enter the  Name  of the list that you want to create and  Description  (Description is optional) ]
p => [  Step 3:  Now, you can see that a list has been created, which has two items,  Title,  and  +Add Column. ]
p => [  Step 4:  Click on the  +Add Column,  you will see that a list of items appears on the screen, select items according to your requirement. ]
p => [  Step 5:  Enter the  Column Name  and click on the  Save. ]
p => [ You will see that a column is created. ]
