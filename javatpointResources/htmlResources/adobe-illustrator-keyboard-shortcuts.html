Keyboard shortcuts

Illustrator becomes more comfortable when we use keyboard shortcuts. Adobe Illustrator allows us to create customized shortcuts according to our requirements. Steps to customize Keyboard shortcuts in Illustrator

Step 1: Go to the Edit menu and select the Keyboard Shortcuts from the drop-down list.

Step 2: The keyboard shortcuts dialogue box will appear then you can choose a set of shortcuts from the set menu.

Step 3: Go to the menu above the shortcut display and choose a shortcut type.

Step 4: After that, do any of the following:

  • Click OK for the activation of the set of shortcuts.
  • Click inside the Shortcut column of the scroll list and type a new shortcut to change a shortcut.
  • To apply the changes you made, click on OK.
