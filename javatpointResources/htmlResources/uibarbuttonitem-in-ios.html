UIBarButtonItem in iOS

In iOS app development, we often use toolbar and tab bar as an integral part of a user interface for ease of navigation. Adding buttons to the toolbar and tab bar has been a good idea since it simplifies the navigation, and most of the important screens in the app can be accessible in one place.

For this purpose, Apple provides us the specialized button for the placement on a toolbar and tab bar represented by UIBarButtonItem class. The syntax for this class is given as follows.

In the iOS app, we can add UIBarButtonItem by using the object library in Interface Builder. However, we can also create the UIBarButtonItem programmatically. We can customize the appearance of buttons by sending the setter messages to UIBarButtonItemApperance. We can also use customized buttons in standard places in a UINavigationItem object (backBarButtonItem, leftBarButtonItem, rightBarButtonItem) or a UIToolbar instance.

In this tutorial, we will discuss how we can add UIBarButtonItems programmatically and by using Interface Builder in an iOS app. We will also discuss how we can add actions to Bar Button Items to perform specific operations when the user performs an event on the buttons.
