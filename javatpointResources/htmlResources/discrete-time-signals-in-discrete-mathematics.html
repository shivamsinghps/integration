We can analyze and represent the discrete-time systems with the help of Exponential sequences. The general form of this is described as follows:

x[n] = Aαn

If there are two real numbers, A and α, in this case, the sequence will be real. If A is positive and 0 < α < 1, in this case, the sequence value will be positive and decrease with increase n in the following way:

If there is a case where -1 < α < 0, then the sign of sequence will be alternate, but it will decrease in magnitude. If there is a case where |α| > 1, then n will be increased, and because of this, the sequence grows in magnitude.
