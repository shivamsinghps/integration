JavaScript DataView.getInt16() method

The JavaScript DataView.getInt16() is an inbuilt method in dataView. It is used to get a signed 16-bit integer(short) number at a specified location.

NOTE: Range of 16-bit integer value is from 0 and 65,535 for unsigned and from ?32,768 to 32,767 for signed integer value.

Syntax

Parameters

byteoffset: The offset, in a byte, from the start of the view where to read the data.

Return value

This method returns a signed 16-bit integer number.
