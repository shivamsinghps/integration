1) What is Tally, and where is it used?

Tally is an accounting and inventory management software package mainly used to record day to day business data of an organization. It is complete enterprise software for small and medium enterprises. The latest version of Tally is Tally ERP 9, which also has several latest subversions released from time to time.

  • Tally software can perform all basic accounting functions of an organization. This software can be integrated with other business applications like Inventory, Finance, Sales, Payroll, Purchasing, etc. Tally software is used to perform the following tasks:
  • It can be used to store all the business transactions of each account in detail.
  • It is the best choice for small-scale enterprises, carries out efficient business transactions, provides accuracy, and saves a lot of time.
  • It can also manage payroll and estimate job costing.
  • It is used to store details and manage inventory items, file a tax return, manage profit and loss statements, prepare a balance sheet, VAT forms, trial balance, cash-flow report, etc.
  • Tally can manage the data over different locations and synchronize it.

  • We can use Tally software to handle financial and inventory management, invoicing, sales and purchase management, reporting, MIS etc.
  • The feature of Tally customization makes the software suitable for distinctive business functions

11) What types of vouchers can be created in Tally ERP 9? What are the shortcuts for creating these vouchers?

Following are the different types of vouchers that can be created in Tally ERP 9:

  • Contra Voucher: This voucher is used to point to the transfer of funds from a bank account to the cash account or cash account to a bank account or in a different bank account of another bank. The shortcut key for creating this voucher is F4.
  • Payment Voucher): This voucher explains the historical payments on each account like payment of salary, rental, etc. The shortcut key for creating this voucher is F5.
  • Receipt Voucher): This voucher describes the transactions that relate to payments received. The shortcut key for creating this voucher is F6.
  • Journal Voucher): This voucher is used to cover the expenses to maintain the organization's running costs. The shortcut key for creating this voucher is F7.
  • Sales/Invoice Voucher: This voucher describes the sales made. It also displays the sale invoice. The shortcut key for creating this voucher is F8.
  • Credit Note Voucher: Entry is made in this voucher when a customer returns the products which weren't correctly dispensed. The shortcut key for creating this voucher is Cntrl+F8.
  • Purchase Voucher): This voucher is for items purchased for running the business like raw materials, etc. The shortcut key for creating this voucher is F9.
  • Debit Note Voucher): This voucher is used for keeping records when goods are returned to suppliers due to any damages. The shortcut key for creating this voucher is Cntrl+F9.
  • Reversing Journals): This voucher is a special entry that gets automatically reversed after the date of the issue. The shortcut key for creating this voucher is F10.
  • Memo Voucher): This voucher is used to mark the unknown expense at a moment which can be converted to an actual sale voucher or deleted. The shortcut key for creating this voucher is Cntrl+F10.

12) What do you understand by deferred tax liability? Deferred taxation is a part of which equity?

Deferred tax liability is a listing on a company's balance sheet that records taxes that are owed but are not due to be paid until a specific future date. It is called deferred tax liability because of the difference in timing between when the tax was accrued and when you have to pay it. Deferred taxation is a part of the owner's equity.

27) What is trial balance, and what is the process to check the trial balance in Tally ERP 9?

Trial balance is mainly a list of all the general ledger accounts within the ledger of a business. This list consists of the name of each nominal ledger account and the value of that nominal ledger balance. Each nominal ledger account contains either a debit balance or a credit balance. All debit and credit balances are kept and compiled into the debit and credit account column. Every organization prepares a trial balance periodically, usually at the end of every reporting period.

Trial Balance is a way to verify that the journal entries are entered correctly in the general ledger. In the debit and credit column, all debit balances should be equal to the total of all the credit balances.
