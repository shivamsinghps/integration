1) What do you understand by Artificial Intelligence?

Artificial intelligence is a technological advancement in computer science that aims to create machines capable of exhibiting human-like behaviour. These intelligent machines can simulate human thinking and decision-making, and are designed to replicate the abilities of the human mind. The term "artificial" refers to the fact that this intelligence is man-made, while "intelligence" pertains to the ability to learn, reason, and solve problems - characteristics that are typically associated with human cognition.

Artificial intelligence technology enables machines to carry out tasks without the need for explicit programming. Rather than pre-programming an algorithm for a specific task, AI machines can be designed to work independently.

7) What are the types of Machine Learning?

The field of Machine Learning can be classified into three primary categories:

  1. Supervised Learning: Supervised learning is a type of Machine learning in which the machine needs external supervision to learn from data. The supervised learning models are trained using the labeled dataset. Regression and Classification are the two main problems that can be solved with Supervised Machine Learning.
  2. Unsupervised Learning: It is a type of machine learning in which the machine does not need any external supervision to learn from the data, hence called unsupervised learning. The unsupervised models can be trained using the unlabelled dataset. These are used to solve the Association and Clustering problems.
  3. Reinforcement Learning: In Reinforcement learning, an agent interacts with its environment by producing actions, and learn with the help of feedback. The feedback is given to the agent in the form of rewards, such as for each good action, he gets a positive reward, and for each bad action, he gets a negative reward. There is no supervision provided to the agent. Q-Learning algorithm is used in reinforcement learning.

8) Explain the term "Q-Learning."

Q-learning is a well-known method utilized in reinforcement learning that is founded on the Bellman equation. In this methodology, the agent endeavors to acquire optimal policies that will assist in maximizing rewards under specific scenarios by learning from previous experiences. By doing so, the agent takes the best actions using the learned policies.

Q-learning is an algorithm wherein the value Q is employed to indicate the desirability of taking each possible action at a given state. The objective of the agent is to maximize the Q value by making the best possible decisions at each state.

14) What do you understand by the reward maximization?

In reinforcement learning, the objective of the learning agent is to maximize the rewards it receives for taking certain actions. This is achieved by the agent receiving positive feedback when it successfully transitions from one state to another, such as receiving a reward for making a good decision. Conversely, if the agent performs an undesirable action, the reward may be subtracted. The overall goal is for the agent to make decisions that result in the highest possible rewards, which is referred to as reward maximization.

15) What are parametric and non-parametric model?

Machine learning models can be classified into two categories: Parametric and Non-parametric models. These models are built using predictor variables which are referred to as parameters. The following is a brief overview of these two types of models.

Parametric Model: The parametric models use a fixed number of the parameters to create the ML model. It considers strong assumptions about the data. The examples of the parametric models are Linear regression, Logistic Regression, Naïve Bayes, Perceptron, etc.

The non-parametric model is a type of statistical model that does not make rigid assumptions about the data and instead uses a flexible number of parameters. These models are well-suited for large datasets and situations where little prior knowledge is available. Some examples of non-parametric models include decision trees, K-nearest neighbor, and support vector machines with Gaussian kernels.

The selection of appropriate hyperparameters is critical to building effective algorithms.

17) Explain the Hidden Markov model.

A hidden Markov model is a statistical technique used to represent probability distributions over a sequence of observations. It assumes that the state of a process generated at a particular time is hidden from the observer, and that the process satisfies the Markov property. This model is particularly useful for analyzing temporal data.

The HMM (hidden Markov model) is a versatile tool with applications in several fields, including but not limited to reinforcement learning and temporal pattern recognition.

21) What is overfitting? How can it be overcome in Machine Learning?

Overfitting is a common problem in machine learning wherein the algorithm tries to capture all the data points, including the noise, resulting in a high variance and low bias in the output. This issue arises due to the algorithm's attempt to fit the model perfectly to the training data, which may not generalize well to new data. Overfitting is a significant concern in machine learning, and techniques such as regularization, cross-validation, and feature selection can help mitigate this problem.

Ways to prevent overfitting in machine learning:

  • Cross-Validation
  • Expanding the quantity of data used in training.
  • Regularization
  • Ensembling
  • Removing Unnecessary Features
  • Halting the training process before it reaches its maximum iterations to prevent overfitting is known as early stopping.

22) Tell one technique to avoid overfitting in neural networks?

The dropout technique is widely used to prevent overfitting in neural networks. It's a form of regularization that involves randomly dropping selected neurons during training. This helps to avoid the network becoming too specialized to the training data.

36) What is reinforcement learning?

Reinforcement learning is a machine learning technique where an agent learns by interacting with an environment and receiving feedback in the form of rewards. No labeled data or supervision is provided to the agent, and it continuously performs actions, changes state, and receives feedback to explore the environment. The feedback can be positive or negative depending on whether the action taken was good or bad.

The following are some of the well-known algorithms used in reinforcement learning:

  • Q-Learning
  • SARSA is an algorithm used in reinforcement learning that involves selecting an action based on the current state, receiving a reward for that action, and moving to a new state. The algorithm includes updating the value function for the current state-action pair based on the observed reward and the estimated future reward.
  • A Deep Q Neural Network (DQN) is a type of artificial neural network used for reinforcement learning. It combines deep learning techniques with the Q-learning algorithm, enabling the network to learn how to select the best action based on input data. The network works by predicting the Q values of each possible action, and then selecting the action with the highest Q value. This approach has been used successfully in various applications, such as playing games and controlling robots.

46) Give the steps for A* algorithm?

The A* algorithm is a well-known form of the Best First Search strategy. It is used to identify the shortest path to the desired endpoint through heuristic and cost-based evaluation. In order to execute this algorithm, the following stages must be completed:

Begin the algorithm by adding the initial node to the OPEN list.

Secondly, it is necessary to verify whether the OPEN list is devoid of any items. If the list is found to be empty, then the failure is returned, and the process is discontinued.

In the third step of the A* algorithm, we need to choose a node from the OPEN list which has the lowest value of the evaluation function (consisting of the cost to reach the node and the estimated cost to reach the goal). If the selected node is the goal node, then we can return a success message and stop the algorithm. However, if the selected node is not the goal node, we need to move on to the next step.

The fourth step of the A* search algorithm involves expanding node n and creating all its possible successors. The expanded node is added to the list of closed nodes. Each successor n' is checked to see if it already exists in the open or closed list. If it does not, the evaluation function is computed for n' and it is added to the open list.

In the fifth step of the A* algorithm, if the current node n' is already present in both the OPEN and CLOSED lists, we need to update the back pointer to reflect the lowest g(n') value.

Step 6 involves repeating Step 2 again.

50) What is a heuristic function, and where is it used?

The heuristic function is utilized in Informed Search algorithms to determine the most promising path towards a goal. By receiving the current state of an agent, the heuristic function calculates an estimation of how far the agent is from achieving its objective. Though not always providing the best possible solution, the heuristic method ensures to find an acceptable solution in a reasonable amount of time. Essentially, the heuristic function estimates the distance between a current state and the desired outcome and is represented by the value h(n), which estimates the optimal path's cost between two states. Additionally, it's important to note that the function's value is always positive.

The criterion for the suitability of the heuristic function is provided as:

h(n) <= h*(n)
