While threads share the resources of the parent process, each thread can still execute its own code independently of the others.

7) What is the main purpose of an Operating System?

The primary functions of an operating system can be categorized into two main categories.

  1. The purpose of system software is to ensure that a computer operates efficiently through the management of its computational tasks.
  2. A platform is created to facilitate the creation and implementation of software programs.

8) What is a socket?

A socket is a communication link established between two applications that allows for data transfer. The points of connection between the two applications are known as sockets.

9) What are the different factors that affect the security of a network?

There are two significant factors that can have an impact on the security of a system.

  1. Unauthorized Access
  2. Viruses

10) What makes a network effective and efficient?

To ensure a network is effective and efficient, there are two main factors that need to be considered.

  1. Performance: Performance can be measured in many ways like transmit time and response time.
  2. Reliability: Reliability is measured by the frequency of failure.
  3. Robustness: Robustness specifies the quality or condition of being strong and in good condition.
  4. Security: It specifies how to protect data from unauthorized access and viruses.
