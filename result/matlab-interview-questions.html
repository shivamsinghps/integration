11) What is memory management functions in MATLAB?

Below are lists of functions related to memory management.

  • Clear: It removes variables from memory.
  • Pack: It saves the existing variables to disk, and then reloads them contiguously.
  • Save: It selectively persists variable to disk.
  • Load: Reloads a data file stored with the save function.
  • Quit: Exits MATLAB and all allocated memory to the system.

16) What are the MEX files?

MEX files are code files written in C or C++ programming languages that can be linked directly to MATLAB during runtime. They provide a way to incorporate C, C++, and Fortran programs into MATLAB. Each MEX file must be compiled for the specific hardware platform on which it will be executed.

20) What Is Pseudo-Random Binary Sequence and Numeric Precision In MATLAB?

A pseudo-random binary sequence is a type of M-file in the Frequency Domain System Identification Toolbox that can be generated for a range of specific lengths, typically between 2^2-1 to 2^30-1. It is also referred to as a Maximum Length Binary Sequence or "mlbs".

The term "numeric precision" refers to the accuracy of numbers represented as double-precision floating-point values. Such values typically consist of 53 significant bits written in binary code, which is roughly equivalent to 15 or 16 decimal digits when converted to base-10 notation.

22) What is a P-code?

Pcode is essentially a pre-compiled and encoded version of the M-file that contains information about the loading time of the function. It is typically not a concern for most M-files as they are parsed only once. One benefit of using Pcode is that it allows us to keep our source code hidden from others, as there is no way to reverse it back to the M-file. Additionally, Pcode is platform-agnostic in nature.

23) Explain about the mentioned tools in MATLAB: who, whos, pi, eps, type.

The "who" command is used to display a list of the variables that you have defined at a given moment in time.

"Whos" provides information on the variables in your code, such as their sizes and additional details.

Pi is a mathematical function that computes the ratio of a circle's circumference to its diameter, often approximated to 3.14159.

EPS is a built-in MATLAB function that provides the smallest floating-point number. It is commonly used in cases where we have a vector with potential zero values in the denominator. Adding EPS to the vector would effectively avoid division-by-zero errors without significantly altering the result.
