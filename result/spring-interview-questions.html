10) What are the different bean scopes in spring?

The Spring framework offers 5 different scopes for beans.

No.ScopeDescription
1)singletonThe singleton scope means that the bean instance is created only once and subsequently returned by the IOC container whenever requested. This is the default scope for beans in Spring Framework.
2)prototypeEvery time a request is made for the bean, a new instance of it will be created.
3)requestA new bean instance will be created for each new HTTP request.
4)sessionA new instance of the bean will be created for every individual HTTP session.

11) In which scenario, you will use singleton and prototype scope?

21) What is JoinPoint?

A JoinPoint is a specific point in your program's code, such as accessing a field or executing a method. It can also include handling exceptions and other events.

22) Does spring framework support all JoinPoints?

Spring framework specifically caters to joinpoints related to method execution and does not cover other types of joinpoints.

23) What is Advice?

Advice is a course of action recommended by someone in regards to a particular situation or problem.

24) What are the types of advice in AOP?

In Spring AOP, there exist five different types of advice.

  1. Before Advice
  2. After Advice
  3. After Returning Advice
  4. Throws Advice
  5. Around Advice

25) What is Pointcut?

Spring AOP uses an expression language called Pointcut to define the join points that will be intercepted by the advice.

26) What is Aspect?

An aspect is a fundamental component in Spring AOP that incorporates advice and joinpoints. It plays a key role in the process of AOP by defining what kind of actions should take place during certain events.

27) What is Introduction?

Introduction refers to the introduction of novel concepts and methodologies that are specifically designed for a particular type or field.

28) What is target object?

The Target Object refers to an object that is influenced by one or more aspects acting as a proxy.

29) What is interceptor?

An interceptor can be considered as an aspect that consists of a single advice within a class.

30) What is weaving?

Weaving refers to the act of connecting one feature to another application.

31) Does spring perform weaving at compile time?

The Spring framework does not perform weaving during compile-time, instead it performs it at runtime.

32) What are the AOP implementation?

There exist three distinct implementations of AOP.

  1. Spring AOP
  2. Apache AspectJ
  3. JBoss AOP
