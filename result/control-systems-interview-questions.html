Control System Interview Questions

Here are some frequently asked interview questions and their corresponding answers for Control System positions that may provide job seekers insights into what to expect during the hiring process.

1) What is meant by System?

A system is a collection of interconnected components or elements that work together to perform a specific function. An example of a system would be an automobile, which consists of various parts and components that work together to provide transportation.

2) What is meant by Control System?

A control system is a collection of mechanical or electronic devices that control, regulate, or direct the operation of a system through a control loop. These systems can be utilized for managing small-scale or large-scale industrial tasks by controlling processes or machinery.

more details..

3) What are the types of control system?

A control system can be classified into two categories.

  1. An open loop control system refers to a type of control system where there is no feedback, meaning that the output is not compared to the desired input. In other words, the input is simply processed through the system and the output is produced without any adjustments.
  2. A closed loop control system refers to a type of control system in which the output is constantly monitored and adjusted based on the desired set point or input. This allows for more accurate control and improved performance.

4) What is open loop and control loop systems?

An open-loop control system is a type of control system where the desired output signal has no effect on the control action. In other words, the control action is not dependent on the output signal that is intended to be achieved. Some examples of open-loop control systems include automatic washing machines and immersion rods.

A closed-loop control system is a type of system where the control action is determined by the desired output. This means that the system can adjust itself to achieve the desired result. Some examples of closed-loop control systems include servo voltage stabilizers, automatic electric irons, and air conditioners.

more details..

5) What is time-invariant System?

Transient time refers to the period required to shift from one state to another state, while transient response refers to the current and voltage values during this period. A system that does not undergo any changes in input and output characteristics over time is known as a Time-Invariant System.

6) What are linear and non-linear systems?

Linear systems are characterized by their ability to demonstrate homogeneity and superposition. In the context of linear systems, superposition is the property whereby applying an input r 1 (t) results in an output c 1 (t), while applying another input r 2 (t) independently yields an output c 2 (t). However, if both inputs are applied together, the resulting output will be the sum of c 1 (t) and c2(t).

r1 (t) + r2 (t) = c1 (t) + c2 (t)

Non-linear systems are those in which the output is not directly proportional to the input, and they lack the properties of superposition and homogeneity. The stability of these systems relies on the initial conditions and input they receive.

When two differential equations have the same order or form, they are referred to as analogous systems.

9) What are the advantages and disadvantages of open loop control System?

Benefits of an open-loop control system.

  • It can be stated that open loop systems are uncomplicated.
  • These are economical.
  • Maintenance needs are minimal and not complicated.

One can mention the drawbacks of using an open-loop control system.

  • One may find that open loop systems are unreliable or imprecise.
  • These systems cannot be trusted.
  • These are slow.
  • There appears to be no feasible way to improve or optimize the current situation or object in question.

10) What are the advantages and disadvantages of closed-loop control System?

Benefits of closed-loop systems.

  • Closed loop systems are considered to be more dependable.
  • The speed of closed loop systems is higher compared to other types of systems.
  • It is possible to deal with multiple variables concurrently.
  • Optimization is possible.

There are some drawbacks to using closed-loop systems.

  • Closed loop systems can be quite costly to set up and maintain.
  • Maintenance is difficult.
  • The installation process of these systems poses a considerable challenge.

11) What are the necessary components of the feedback control system?

To create a feedback control system, you will need four essential components: an open loop system for processing, a feedback path, an error detector, and a controller.

12) What is the feedback in the control system?

Feedback refers to the process in a control system whereby the output signal is sampled and compared with the input signal. The proportional signal obtained from this comparison is then fed back to the input, enabling automatic correction of any errors. This ensures that the output signal meets the desired specifications.

Gain margin is the gain which varies before the system becomes stable because if we continuously increase the gain up to a certain threshold, the system will become marginally stable, and if the gain varies further then it leads to instability. Mathematically, it is the reciprocal of the magnitude of the G(jω)H(jω) at phase cross-over frequency.

Signal flow graphs provide a visual representation of the relationship between variables for a set of linear equations. They are graphical in nature and do not require any specific reduction technique or process to be implemented.

17) What is the basic rule for block diagram reduction technique?

In block diagram reduction, a key principle is that any modifications made to the diagram should not affect the system's input-output relationship. This means that we can make changes without altering the overall functioning of the system.

18) What is an order of a system?

The order of a system is determined by the highest derivative in the system's equation or the highest power of 's' in the denominator of its transfer function.

19) What is the resonant peak?

The Resonant Peak is the highest value attained by the closed-loop transfer function. A higher resonant peak indicates greater overshoot in the transient response of the system.

20) What is the cut-off rate?

The cut-off rate of a system refers to the slope of the log-magnitude curve at or near the cut-off frequency. This measure signifies the system's capability to discern between the signal and noise components in a given signal.

21) What is phase cross-over frequency?

The phase crossover frequency refers to a frequency at which the phase of the open loop transfer function reaches 180 degrees.

22) What is Phase Margin?

The amount of phase lag needed at the gain crossover frequency to push a system close to instability is referred to as the Phase Margin.

23) What Is Pole Of The System?

In complex variable functions, the point where the function F(s) blows up and becomes infinite is referred to as the Pole of the function F(s).

24) What Is Zero Of The System?

The point where a function F(s) of complex variables equals zero is referred to as its Zero.

25) What Is The Use For Cable Entry In Control Room?

In the event of an emergency, such as a fire or explosion in a plant, it is essential to prevent it from spreading to the control room. Multiple Cable Transient (MCT) blocks are employed to achieve this, and control rooms are constructed in non-hazardous areas.

26) What is the effect of positive feedback on the Stability of the systems?

Positive feedback is a type of feedback that amplifies the error signal and pushes the system towards instability. Because of this, it is not widely used in control systems. However, positive feedback may be utilized in minor loop control systems to amplify internal signals and parameters.

27) What is Servomechanism?

A servomechanism is created when a servomotor is paired with either a rotary encoder or a potentiometer. This pairing allows for position feedback with the potentiometer providing an analog signal and the encoder providing feedback on both position and speed.

28) Where is Servomechanism used?

A servomechanism is utilized to regulate the position of a mechanical device through the output it receives. This technology is commonly used in control systems.

The Servomechanism has multiple applications, including its use in Governor value position control mechanisms in power plants and robotic hand movement. This technology is used to process turbine speed using transducers, which convert the speed into mechanical movement of the value. While this mechanism was initially used with mechanical controls, modern systems make use of electronic controls that employ power Thyristors.

29) How many types of instrument cables are there?

There are different kinds of cables that can be used for musical instruments.

  • IS cables
  • NIS cables.
  • IS and NIS refer to two different classifications for equipment used in potentially explosive environments. IS stands for Intrinsic Safety, which refers to equipment that is designed to operate without emitting any sparks, heat, or other sources of ignition that could cause a potential explosion. NIS, on the other hand, stands for Non-Intrinsic Safety, which refers to equipment that does not have these safety features and could potentially cause a hazardous incident in explosive environments.
  • The selection of the cable type depends on the nature and severity of the hazards present.

  3. Trough Cable Tray
  4. Channel Cable Tray
  5. Wire mesh cable tray is a type of cable management system made from a mesh of interconnected wires. It provides an efficient and cost-effective solution for organizing cables and wires in various settings. The wire mesh design allows for easy installation, flexibility, and good ventilation for the cables and wires.
  6. The following is an original description of the single rail cable tray: The single rail cable tray is a type of cable management system designed to hold and organize electrical cables. It features a single, centrally located rail upon which the cables can be mounted, making it easy to add, remove or adjust cables as needed. This type of cable tray is often used in industrial settings where large amounts of cables and wires need to be managed efficiently. It is an effective way to prevent cable damage and ensure the safety of people working in the area.

Before installing any cables, it is important to carefully assess the location and the available space where the cables will be laid. This will ensure that the installation is suitable for the site conditions and that the cables will be properly secured.

One needs to consider the number of cables required and the occupancy in the cable tray before selecting its size. Cable trays are available in varying sizes such as 80, 150, 300, 450, 600, and 900.

33) What is the cut-off Rate?

The cut-off rate refers to the steepness of the slope around the frequency where the log-magnitude curve starts to decrease. It denotes the system's capacity to differentiate the signal from noise.

34) Enlists the applications of Sampled Data Systems?

A system that deals with sampled data finds its use in several applications.

  • High-speed tinplate rolling mills rely on quantized data for control.
  • Electric drives can be either digitally controlled or pulse controlled. Digitally controlled electric drives use digital signals to manage the electric motor, while pulse controlled drives control the motor using pulses of current.
  • Numerical control is a system used in machine tool operations to automate the process. It involves the use of programmed instructions in the form of numbers and symbols to control the movement and actions of machines such as lathes, mills, and routers. This technology has revolutionized manufacturing processes by increasing efficiency, precision, and flexibility while reducing labor costs and error rates.

Pulse code modulation (PCM) is utilized in significant systems that make use of telemetry links reliant on PM translational data.

35) What are DCS and PLC?

DCS and PLC are two types of control systems used to handle the input/output processes in various industrial applications. The DCS stands for Distributed Control System, while the PLC refers to a Programmable Logic Controller.

36) What are stable systems?

A stable system is one where the roots of the characteristic equations are located on the right half of the "S" plane.

37) What are marginally stable systems?

A marginally stable system is a type of system where all the roots of the characteristic equation are located on the imaginary axis of the "S" plane. This means that the system is neither stable nor unstable, but remains on the verge of instability.

38) What are unstable systems?

Unstable systems refer to those whose characteristic equation roots are situated on the left half of the 'S' plane.

39) What is Routh Hurwitz Stability Criterion?

The Routh Hurwitz criterion provides a condition for the stability of a system. It states that a system is considered stable only if all the roots in the first column have either strictly positive or negative real parts. If the signs in the first column are not the same, the number of times the sign changes within the first column must be equal to the number of roots with a positive real part in the characteristic equation.

40) What is an Automatic Controller?

Automatic Controllers are electronic devices that compare the real-time output of a plant with a predetermined value. They are responsible for regulating and controlling the system to minimize or even eliminate any deviation from the desired output. These systems function by measuring the difference between the desired and actual values and taking corrective action to bring the deviation to zero or a minimum value.

41) What is the Control Action?

The control action refers to the specific method used by an automated controller to generate the control signal.
