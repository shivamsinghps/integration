Unix Interview Questions

Here are some common Unix interview questions and answers.

1) What is Unix?

UNIX is an operating system that has been designed to enable efficient multi-tasking and multi-user functions that can run on various hardware platforms. The operating system is written in C programming language and was created by a group of developers that included Ken Thompson, Dennis Ritchie, Douglas McIlroy, and Joe Ossanna.

2) List the distributions of UNIX.

There are numerous distributions of UNIX available, such as AIX, HP UNIX, BSD, and Solaris UNIX, among others.

3) List some features of UNIX.

UNIX has various characteristics that distinguish it from other operating systems.

UNIX has the ability to support multiple users on the system by providing them with their own login and workspace. It is equipped with features that enable it for multi-user systems, making it ideal for environments where several people need to use the same computer.

UNIX enables the operation of multiple applications simultaneously, creating a multitasking environment.

4) What are the core concepts of UNIX

The fundamental principles that underlie the UNIX operating system can be summarized as follows.

  • Kernel- The kernel is also known as the heart of the operating system. Its fundamental role is to interact with the hardware and also monitor major processes like memory management, file management, and task scheduling.
  • Shell- It is also called command prompt, it connects the user to the operating system, whatever is typed by the user is translated into the language understood by the command prompt, and then the corresponding actions are performed.
  • Commands and Utilities- Many built-in commands help the user perform day to day activities.mv,cat,cp,and grep etc. Some of the examples
  • Directories- Every bit of data is stored in files, and these files are stored in directories, these directories combine to form a tree-like structure.

5) What is a UNIX shell?

The UNIX shell acts as a mediator between the user and the UNIX OS, enabling communication with the server. Although it is not a component of the kernel, it can interface with it.

6) What is filter?

In computing, a filter is a software application that receives data as input and performs actions on that data to generate a new result as output. The input is usually obtained from a standard input source and the output is typically sent to a standard output source.

7) What are the devices represented in UNIX?

One of the fundamental concepts in UNIX is that devices are represented as files located in the /dev directory. This means that each device has its own corresponding file in the system.

8) Is there any method to erase all files in the current directory, along with its all sub-directories, by using only one command?

Sure, I can rephrase that for you. You can use the command "rm-r*" to achieve that goal.

In the given scenario, a command called "rm" is utilized to remove files. This command comes with an option "-r" that enables deleting directories along with their internal data. Additionally, an asterisk (*) is used for selecting all entries that need to be erased.

9) What is necessary before running a shell script from the terminal?

To run a shell script, first, you need to grant executable permission to the script using the "chmod" command in the UNIX terminal.

10) How to terminate a shell script if statement?

In shell scripting, to end an if statement, the keyword "fi" is used.

11) Write down some common shells with their indicators?

  • Sure, here is a paraphrased version: The Bourne shell, often denoted as sh, is a command-line interface used in Unix and Unix-based operating systems. It is a command processor that runs tasks and programs on the system. The Bourne shell was developed by Stephen Bourne in the 1970s and is one of the most widely used shells in Unix-based systems.
  • The C Shell, commonly known as csh, is a Unix shell that was developed by Bill Joy as a replacement for the Bourne shell (sh).
  • The Bourne Again Shell, commonly known as "bash," is a command language interpreter used on Unix and Linux operating systems. It was designed to be compatible with the original Bourne shell, while also adding additional features and functionality.
  • TCsh, also known as the enhanced C Shell, is a Unix command-line shell. It provides several features not found in the original C Shell, such as command-line completion, spelling correction, and a more powerful scripting language.
  • Z Shell, also known as zsh, is a Unix command-line interpreter shell that provides advanced features and customization options compared to the traditional Bourne Shell.
  • Ksh is a type of shell, specifically the Korn Shell.

12) What are the main features of Korn Shell?

  • Arrays
  • Job control
  • Command Aliasing
  • String manipulation ability
  • Built-in integer arithmetic

13) What is the difference between cat command and more command?

To view file contents on the terminal, the cat command is used, while the more command operates like a pager by displaying the screen content one page at a time, making it useful in cases where the file is too large to view in one screen.

14) Which command is used to restrict incoming messages?

The command "mesg" is utilized for the purpose of limiting or preventing the receipt of incoming messages.

15) Which command is used to kill the last background job?

One way to terminate the last background job is by using the command "kill $!".

16) Which data structure is used to maintain the file identification?

The "inode" is a crucial data structure that is responsible for keeping track of file identification. Every file has its own unique inode number which is stored in an inode data structure.

17) What a pipe?

A vertical bar or a pipe symbol (|) is utilized to connect multiple commands in a Linux shell. The data output from the first command is then transmitted to the subsequent command via a pipe.

18) What are the links and symbolic links in a UNIX file system?

A link is an alternative name given to a file. Its purpose is to allow multiple names to be associated with a single file, however, it is not possible to use links to assign more than one name to directories or link filename between different computers.

Symbolic links are special files that serve as pointers to other files. When we perform operations on the symbolic link, they are directed to the file to which it is pointing. Unlike hard links, there are no restrictions on the number of symbolic links we can create, hence making them more flexible.

19) Explain system bootup in UNIX.

In UNIX, the initial process that occurs upon pressing the power button is the system bootup. The first step involves the activation of the BIOS, which then verifies that all hardware components are functioning normally. Following this verification process, the system prompts the user to authenticate themselves.

20) How to change the password in UNIX operating system?

One way to modify the password in a UNIX operating system is by following certain steps.

  1. Please enter the command "passwd."
  2. To begin changing your password, a prompt will appear on the screen requesting your current or default password. Please enter your current password in the designated field.
  3. Once you confirm your existing password, the terminal will request you to input a new password.
  4. To change your password, type in the new password two times. Once you've done this, your password will be successfully updated.

21) How to list directories in UNIX?

The "ls" command is a useful tool for listing directories in a command prompt environment. Additionally, there are various "ls" commands that can be utilized for different purposes.

ls -a In the Linux operating system, there are files that are hidden, indicated by the prefix of a dot symbol, and are not visible in the regular directory. To view these hidden files, the command line prompt "ls -a" can be used to display the full list of all files, including the hidden ones, in the current directory.
ls -l The displayed content will be in a format where the list is shown in a lengthy sequence.
ls -lh The command "ls -lh" can be used to display file sizes in a human-readable format. This is helpful as it can be difficult to understand the size of a file when it's presented in bytes. Instead, the command displays the size in terms of Mb, Gb, Tb, and so on.
ls -lhS To show your files arranged according to their size in descending (largest first), you can execute the command "ls -lhS".
"List files and directories in long format and with block size specified by replacing [SIZE] in the command." Possible rephrased version: Use the 'ls' command to display detailed information about files and directories, and set the block size by replacing the placeholder [SIZE] in the command. This command is used for showcasing files in a particular size format. You can specify the size you need within the square brackets after the command.
ls -d */ This command is utilized to exhibit solely the subdirectories.
One option to list the contents of a directory in the terminal is by using the command "ls". This command has various options that can modify the output. Two of these options are "-g" and "-lG". Both options display the contents of a directory in a long format, showing various information such as the file's size, owner, permissions, and group. The difference between the two options is that "-g" excludes the name of the owner from the output, while "-lG" includes it. You can exclude columns related to group information and owner using this method.
ls -n The purpose of using the "-n" option in certain Unix commands is to display the numeric group ID and owner ID rather than their corresponding names. This can be useful in cases where displaying the numerical values is more important than displaying the names.
ls --color=[Value] To output a list in a colored or uncolored format, the "print" command can be used.
ls -li One way to modify the given command is to output the index number of the file in the first column.
ls -p The slash (/) line sign is utilized in marking directories for easy identification purposes.
ls -R The output of the command will show the contents of any subdirectories included.
ls -lX The function will categorize the files based on their file extensions and group them together in the displayed list.
ls -lt The list will be organized in such a way that the file which has been modified most recently will appear at the top.
ls ~ The output displays the items or files stored in the user's home directory.
ls ../ The command displays the files and directories that are located in the parent directory.
ls --version The code snippet is used to verify the version of the "ls" command.

22) How to check the date in UNIX?

To show the current date in a UNIX system, you can utilize the "date" command within the command line interface.

23) How to log out in UNIX?

To end your session in UNIX, use the "logout" command in the command prompt.

24) How to perform a system shutdown in UNIX?

To initiate a system shutdown in a UNIX operating system, there are specific commands you can use.

  • halt
  • init 0
  • init 6
  • power off
  • reboot
  • shutdown

25) How many types of files are there in UNIX?

UNIX has three main types of files.

  • An ordinary file is a type of file that stores data, text, or program instructions.
  • Directories consist of different types of files, including regular files and unique files.
  • Special Files are a category of system files that grant exclusive access to hardware components like hard drives, CD-ROM drives, and others. They are crucial for the normal operation of computer systems.

26) What are hidden files in UNIX?

In UNIX operating systems, there are certain files that are not visible in the regular file manager. These files are referred to as hidden files and can be identified by the dot (.) that appears before their file name.

Some files on a computer are hidden and not easily visible to the user. These files are typically named with a period ("."), which makes them less noticeable. Such files are commonly found on operating systems like macOS, Linux, and Windows. Some examples of these hidden files include configuration files, system files, and files used by applications.

  • .profile
  • .kshrc
  • .rhosts
  • .cshrc

27) What is the difference between a single dot and double dot in UNIX?

The symbol "." (single dot) represents the current directory.

The notation "double dot" is used to indicate the parent directory in a file system.

28) How to create files in UNIX?

In UNIX, it's easy to create new files by using the vi editor. It's a simple process that can be accomplished by the user.

One way to create a new file is by using the command prompt and typing "vi filename". Another option is to use the "touch" command to create a file with zero bytes.

29) How to display the contents of a file?

To display the contents of a file, the user can enter a command in the command prompt using the cat command followed by the name of the file. The format of the command should be followed as prescribed.

$ cat filename

Where the cat is the command to view contents of the file specified by the filename. Also if you want the line number to be displayed along with the content, you can use cat command with option -b.

30) How to calculate the number of words in a file?

The following command can be used to determine the word count for a file:

$ wc filename

The instruction asks to use the 'wc' command to determine the word count of a specific file named 'filename'.

31) How to create a blank file in UNIX?

To create a blank file, one can utilize the touch command. The format of the touch command is as follows:

$ touch filename

32) How to know the present working directory in UNIX?

One way to find out the current working directory is by entering a specific command in the terminal.

$ pwd

33) How to know the information about a file?

To acquire details of a file, utilize the subsequent command.

$ file filename

34) How to change the directory in UNIX?

The way to modify the directory in the terminal is through the use of the cd command. This command enables you to set a new directory as the current directory from the terminal window.

$ cd directory-name

35) How to move files from one directory to other in UNIX?

The UNIX command 'mv' is utilized to relocate a file from one directory to a different location.

One way to move a file using a Unix/Linux command line is to use the "mv" command followed by the file name and destination path.

36) How to copy files from one directory to other in UNIX?

The cp command is used in the UNIX operating system to make a copy of a file and place it in a different directory. Its format is as follows:

The copied content presents a command that can be used to copy a file or directory from one location to another: "cp -r source filename destination file name."

The option "-r" is frequently employed to duplicate all files and folders within a directory and its sub-directories in a way that is recursive.

37) How to remove files in UNIX?

If you want to delete files, you can make use of the rm command, which allows you to remove files. Here's the correct syntax for using the rm command to delete files.

One way to remove a file in a Unix or Linux operating system is by using the "rm" command followed by the name of the file you want to delete.

One way to delete all subdirectories recursively is by using the '-r' option with the 'rm' command.

38) How to make a new directory in UNIX?

You can create a new directory by using the mkdir command.

$ mkdir <directory-name>

39) How to remove the directory in UNIX?

One way to delete a directory is by using the command "rmdir". Here's how to use it:

$ rmdir filename.
