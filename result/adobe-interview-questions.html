5) Differentiate between new and malloc()?

Dynamic memory allocation can be achieved using two techniques, namely new and malloc(). Although both techniques are functionally similar, there are several differences between them. These differences are outlined below.

  • In C language, there is an operator called "new" which is used for memory allocation. On the other hand, "malloc()" is a memory allocation function.
  • When we use the "new" operator to allocate memory for an object, it also calls the constructor of the object. However, when we use "malloc()" to allocate memory, it doesn't call the constructor of the object.

1) What are the different layers in TCP/IP models? Explain the protocols at each layer?

The TCP/IP model is an abbreviation for Transmission Control Protocol/Internet Protocol. It's a streamlined and practical version of the OSI model that is widely used in modern network architecture. The TCP/IP model is made up of four primary layers.

  • Application Layer
  • Transport Layer/Host-to-host layer
  • Network Layer
  • The Link Layer or Network Access Layer refers to the lowest layer of the OSI model, responsible for facilitating communication between devices on the same network.

1. Link Layer:

The data link layer is a crucial component of computer networking that manages the transmission of data through the network. It involves various hardware components such as twisted pair, coaxial cable, or optical fibre, and is responsible for the physical flow of data. At this layer, data is transmitted in the form of frames and bits.

This layer of the OSI model makes use of protocols such as Ethernet and token ring.

2. Internet Layer:

The Internet layer is positioned above the Link layer. Its main function is to package data into IP datagrams or data-packets, which are then transmitted across the network from source to destination host. These datagrams contain both the source and destination addresses, which guide their route through the network.

At the network layer, several protocols are utilized, including:

  • IP (internet protocol): This protocol is responsible for forwarding the packets from source host to destination using an address which is consisted by packet header. IP has two main versions IPv4 and IPv6.
  • ICMP (Internet Control Message Protocol) is a communication protocol used in computer networks. It is responsible for handling network errors and providing feedback to network devices.: This protocol is mainly responsible for informing the problem in the network.
  • ARP (Address Resolution Protocol) is a communication protocol used in computer networks. It helps to resolve the network address (IP address) of a device to its corresponding physical address (MAC address). ARP works by broadcasting a request message, and the device with the matching IP address responds back with its MAC address. This information is stored in a cache on each device, which speeds up subsequent communication between the devices.: ARP protocol is an essential protocol of the TCP/IP model, and it is responsible to resolve an IPv4 address to physical address.

The TCP/IP model comprises of three layers, the first of which is the network layer that handles data transmission over the physical network. The second layer, called the internet layer, focuses on addressing and routing packets across different networks. Lastly, the transport layer is responsible for ensuring reliable and error-free communication between the source and destination hosts.

The second layer of the OSI model is composed mainly of two protocols:

Transmission Control Protocol (TCP) is a trustworthy method for ensuring accurate communication between the source and receiving end. It requires a connection to be established before information can be exchanged, making it a connection-based protocol.

UDP (User Datagram Protocol) is a protocol that offers less reliability compared to TCP. It can be used in situations where reliable communication is not required, as it is a connectionless protocol.

The TCP/IP model consists of four layers. The first layer is the link layer which is responsible for providing communication between connected devices on the same network.
