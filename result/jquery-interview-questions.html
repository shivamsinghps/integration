1) What is jQuery?

jQuery is a popular and efficient JavaScript library used for client-side programming. The library is recognized for its lightweight nature, high-speed performance, and feature-rich capabilities. It is cross-platform compatible and supports multiple web browsers. With its capabilities, jQuery has transformed web development by providing a simpler and faster method for JavaScript programming. Prior to jQuery, JavaScript programming involved writing extensive code, particularly for basic functions. jQuery has revolutionized website interactivity and provided developers with an array of tools to enhance website designs.

2) Is jQuery a programming language?

jQuery is not a programming language, but rather a collection of JavaScript code that is well-crafted. Its primary functions include navigating documents, handling events, interacting with Ajax, and animating elements on web pages.

3) What is the difference between JavaScript and jQuery?

JavaScript and jQuery are two different things. JavaScript is a programming language commonly used for creating interactive websites while jQuery is a library built for JavaScript. The purpose of jQuery is to simplify the use of the JavaScript language, making it easier for developers to code various website functions. Essentially, jQuery is a tool that enhances the capabilities of JavaScript.

4) Is jQuery replacement of JavaScript?

Although jQuery is written in JavaScript and interacts with JavaScript and HTML, it is not a replacement for JavaScript itself. Rather, jQuery is a separate and lightweight library that builds upon JavaScript.

7) What are the effects methods used in jQuery?

Here are some techniques utilized in jQuery to achieve various visual effects:

  • The function 'show()' is responsible for exhibiting or displaying the particular elements that have been chosen.
  • The hide() function conceals the elements that have been matched or selected.
  • The toggle() method is used to display or hide elements that match a given pattern. Put simply, using toggle() alternates between calling the hide() and show() methods.
  • The fadeIn() method is used to display the selected elements by slowly increasing the opacity until it becomes completely opaque. In simpler terms, it is used to create the effect of the elements fading in.
  • The function fadeOut() is used to gradually make the selected elements fade to transparent, essentially hiding them. This means that any matching elements will slowly disappear from view.

21) What is a CDN?

A CDN is short for Content Delivery Network also known as Content Distribution Network. It is made up of a network of servers located in various data centers across the internet. Its purpose is to provide files to users at a higher bandwidth resulting in faster loading times. There are some companies that offer free public CDNs.

  • Google
  • Microsoft
  • Yahoo

22) What is the goal of CDN and what are the advantages of using CDN?

CDNs are designed to offer superior content availability and performance to end-users.

Benefits of utilizing Content Delivery Network (CDN):

  • One benefit of using caching techniques is that it decreases the burden on the server.
  • Using a Content Delivery Network (CDN) for jQuery framework can result in faster loading time due to better utilization of bandwidth. By hosting jQuery on a CDN, it can be loaded faster which can significantly improve the performance of web applications.
  • When a website is using jQuery framework from a Content Delivery Network (CDN) and a user frequently visits the site, the framework will be cached for faster loading times in the future.

27) What is the difference between the ID selector and class selector in jQuery?

The ID selector and class selector are similar in JavaScript and HTML as they are in CSS. The ID selector identifies elements by their unique ID, while the class selector identifies a group of elements by a shared class.

You use an ID selector to select just one element. If you want to select a group of elements, having the same CSS class, use class selector.
