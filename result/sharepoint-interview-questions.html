SharePoint Interview Questions and Answers

Below are commonly asked questions in SharePoint interviews along with their corresponding answers.

1) What do you know about SharePoint?

SharePoint is a software platform that enables users to create websites and manage content. It offers a variety of features that enhance web application development, knowledge sharing, collaboration, and information management. One of its significant advantages is that it can be accessed from anywhere around the globe as long as there is an internet connection. This service was developed by Microsoft.

3) Why do we use SharePoint?

SharePoint is a popular software that is widely used by many organizations for various purposes.

  • Share documents ? SharePoint allows you to store and share documents with multiple employees at the same time.
  • Business Intelligence ? SharePoint allows you to organize the data to make better decision-making relationship between different queries based on the graphs and charts.
  • Collaboration ? Through SharePoint, employees can collect information from different sources in a central location. Therefore, SharePoint helps you to increase the use of technology.
  • Document Management ? SharePoint offers better document management capability, like adding the permissions such as who can access the particular document and who can?t access these documents.
  • Websites ? SharePoint allows you to build and manage public websites. As it is a CMS (Content Management System), so it also allows non-technical users to create, modify, and delete the websites easily.

7) What is WSP?

A WSP, which stands for Web Solution Package, is a type of file that contains a variety of resources, including site definitions and images. It provides a convenient way to mix these features with a website.

What is the team site in SharePoint?

A team site is a platform designed for individuals to save and work on files together. It allows them to create and organize lists of information in a shared environment. The site is composed of web pages, a pre-installed document library where files can be stored, lists used for data management, and web parts.

19) What is the picture library, and how to create a picture library in SharePoint?

The image gallery function of a website lets you save and modify images which can be utilized within your website or shared with others.

To create a picture library, there are several steps that need to be followed.

To start adding a new app to your website, first, navigate to the Home page. From there, locate the +New button and click on it. Then, choose the option that says "App" from the list of choices that will appear in the drop-down menu.

In the second step, you need to look for a library of images. Look for the option that says "picture library" and click on it.

To begin creating a picture library in SharePoint, you need to open a new site or select an existing one. Next, navigate to the "Site Contents" page to find the "Pictures" option. Click on it and then select "Picture Library" from the dropdown menu. After that, provide a name for the picture library and click on the "Create" button.

After clicking on the Create button, a new picture library will be generated.

23) What is the difference between CustomMasterUrl and MasterUrl?

CustomMasterUrl is a feature that enables the customization of the layout of admin pages.

The MasterUrl is a useful tool for modifying the appearance of multiple pages viewed by end-users. It allows for universal formatting changes across a website's pages.

24) What is a Site Collection?

A site collection in SharePoint is a collection of related websites. SharePoint Server offers two kinds of site collections, which are host-named site collections and path-based site collections.

25) What is the page layout?

Page layouts refer to the preset designs that determine the arrangement, fields, and web components on a page. They enable users to define how pages appear on their websites.
