Silverlight Interview Questions

The following are some common questions and answers that you may come across during a Silverlight job interview.

1) What is Microsoft Silverlight?

Microsoft Silverlight is a tool used for developing and deploying interactive internet and mobile applications, as well as media experiences. It is an open-source development tool that provides users with a platform for creating engaging and immersive user experiences.

2) Is Silverlight compatible with multiple browsers and operating system?

Yes, Silverlight works with various web browsers, devices, and operating systems. It is a plug-in that utilizes the .Net framework and is offered for free.

3) Which language is used to design the layout in Silverlight?

Silverlight employs a language named XAML to create its interface design. XAML, which stands for Extensible Application Markup Language, is an XML-based declarative language developed by Microsoft.

4) What is the different development tools used in Silverlight?

The Microsoft Visual Studio is the primary tool used with Silverlight applications. Developers write applications in the .NET language and use all of the associated tools that come with it. Microsoft Expression Blend is a specialized tool for designing and developing the user interface of Silverlight applications. Debugging errors and ensuring that applications run smoothly is typically done using Visual Studio. Silverlight applications are packaged as a zip file, which contains a .XAP file that includes one or more .NET managed assemblies stored as .DLL files. Additionally, the AppManifest.XAML file contains the entry point and essential classes and assemblies for the application.

5) What is RIA?

RIAs are internet applications that are enriched with advanced features. They are designed with built-in AJAX support, multimedia components, animations, layouts, and more. These features distinguish them from regular web applications.

Silverlight can be referred to as a type of RIA.

9) Is ADO.Net objects supported in Microsoft Silverlight project?

Silverlight projects do not have the capability to utilize standard ADO.NET objects including DataTable, DataSet, DataColumn, and connection providers like SqlConnection and OledbConnection objects.

10) Explain the architecture of Silverlight?

Silverlight's framework consists of three main components:

  1. Core presentation framework: This framework includes the components like data binding, vector graphics, text, animation, images for presenting various features.
  2. The .NET framework is an essential component for building Silverlight web applications. It provides a system of libraries and tools for developers to create robust and dynamic applications. With its extensive collection of functionalities, the .NET framework offers a convenient platform for building and deploying Silverlight solutions. It includes the libraries and components like XLINQ, XML serialization, Syndication, base class libraries, networking and common language runtime.
  3. Updater and Installer: It is a control for installation and provides automatic updates.

11) What are the features of Silverlight plug-ins?

The Silverlight plug-ins offer a range of features, including:

  • Presentation Engine
  • XAML Parser
  • .NET Parser
  • .NET Framework
  • Media Features
  • Browser Interaction
  • Downloader

12) What is .xap file in Silverlight?

A .xap file is used for Silver applications and is a compressed file that typically contains the AppManifest.xaml, compiled output assembly of the Silverlight project, and other resources for the Silverlight application. During the process of creating a project in Visual Studio, two files are created, namely the App.xaml and Page.xaml.

13) How would you use a .xap file?

The purpose of a .xap file is to store and distribute the assemblies and resources of a managed code application that is designed to run within the Silverlight 2 browser plug-in. This file type enables smooth and efficient deployment of Silverlight applications.

14) Which tools are required to build Silverlight applications?

To develop Silverlight applications, there are two essential tools that are necessary.

  1. Microsoft Expression Studio: This tool is used by web designers to create rich visual elements for Silverlight applications with enhanced visual contents and graphics.
  2. Microsoft Visual Studio: This tool is used by programmers whose application is based on logic or requires programming. It permits programmer to build Silverlight applications in any .NET language like C#, Visual Basic.NET, etc.)

17) What is the difference between WPF (Windows Presentation Foundation) and Microsoft Silverlight?

WPF includes all the functionalities of Silverlight, making it accurate to say that Silverlight is a smaller version of WPF.

Silverlight and WPF are both technologies used in application development. While Silverlight is optimized for creating internet applications with advanced features, such as those found in Adobe Flash, WPF is designed to create graphic-rich applications specifically for desktop platforms.

18) How would you perform event handling in Silverlight?

Silverlight 1.0 enables event handling through JavaScript, whereas Silverlight 2.0 primarily employs C# to accomplish the same task.

19) What is the name of Linux version of Silverlight?

Moonlight is the name given to the Linux distribution of Linux.

20) What do you know by Silverlight runtime?

The Silverlight runtime is an essential component that plays a pivotal role in downloading the .xap file from the server. It enables users to view and enjoy the output of the Silverlight content on the webpage with ease.

22) What is isolated storage in Silverlight?

The term "isolated storage" refers to a virtual file storage mechanism used in the context of Silverlight. This system enables the storage of data within a concealed folder on a computer system. Essentially, the data is segregated into two different segments.

  • Section1: It contains the administrative information like disk quota.
  • Section2: It contains the actual data.

Every Silverlight application is assigned a dedicated storage space which currently has a limit of 1 MB per application.

23) How can you see a Silverlight application in full-screen mode?

To display the Silverlight application in full screen mode, you can utilize a specific function or command.

Application.Current.Host.Content.IsFullScreen = true;

24) Which types of video files can be supported in Silverlight?

Silverlight is capable of supporting a variety of video file formats, including MP3, Window Media Audio Format (wma, wmv 7-9), and VC-1 formats.

25) What is Deep Zoom in Silverlight?

Silverlight has a built-in feature called Deep Zoom, which enables users to zoom in and out of an application.

26) Is it possible to pass parameters to Silverlight controls from ASP.NET pages?

Certainly. By using InitParameters, you can transmit parameters from your ASPX pages and HTML pages to the Silverlight controls. This allows for the passing of information between the two components.

The XAML page user control includes an attribute called InitParameters that can be used to pass key-value pairs. This attribute can accept different sets of string values. It is possible to set a value for this attribute from ASPX pages.

28) What is the usage of ClientBin folder?

The ClientBin directory is essential in ASP.NET applications as it enables the use of Silverlight features. Within this folder exists the .xap file containing the necessary reference points for Silverlight libraries, which can be utilized by ASP.NET applications.
