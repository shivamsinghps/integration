Cognizant C Programming Interview Questions

1) What is a pointer? Define it.

A pointer is a type of variable that stores the memory address of other variables, allowing us to indirectly access and manipulate their values. It is commonly used in programming to reference and modify certain variables and their associated data.

2) What is a dangling pointer in C?

A dangling pointer is a type of pointer that points to an invalid object or nothing at all. This issue can occur when the pointer is in the stack but not in the memory heap. A common example is a char *p = NULL. If the pointer is deallocated without first allocating space, it can result in a segmentation fault. In other words, attempting to use a dangling pointer that has not been properly allocated can lead to errors and program crashes.

A situation called dangling pointers can occur when an object is destroyed while still having an incoming reference, causing the pointer to continue pointing to the memory location of the previously allocated memory. This happens when the data of the pointer is not updated during the object deallocation process.

3) What is a memory leak in a heap?

The scenario when there is unoccupied memory space within a heap, but no variable within the stack is directed towards that specific area.

4) What is a data type?

A data type is a classification of the type of data that can be assigned to a variable. It determines the range of values that can be stored in the variable. For instance, in the C programming language, the "Int" data type can store values between "-32768 to +32768".

5) What is the size of the integer data type?

The size of it can either be 2 bytes or 4 bytes.

Malloc is a function that is commonly utilized for dynamic memory allocation in programming. It is responsible for allocating the necessary number of bytes requested by the program and returning a pointer to the first byte of the allocated space.

C language is classified as a middle-level language since it acts as an intermediary between low-level machine languages and high-level programming languages.

The C programming language is a type of language that is considered to be more approachable than machine-level languages. However, it does not encompass all of the features that are available in high-level programming languages and is therefore classified as a middle-level language.

14) What is the difference between structured oriented, object-oriented and non-structure oriented programming language?

A structured oriented programming language is a type of programming language that focuses on using structured programming techniques to build logical and efficient programs. It emphasizes the use of control structures such as loops, conditionals, and functions to create easily readable and maintainable code.

In this type of language, large codes fragmented into small programs called functions. The main focus is on procedures and functions that operate on values Data moves without any restrictions around the systems from one procedure to another Program structure follows "Top-Down Approach." Example: C, Pascal, ALGOL, and Modula-2

An object-oriented programming language is a type of programming language that uses the concept of objects, which are instances of classes, to design and develop software systems.

In this type of language, programs fragmented into objects The main focus is on the data/values that are manipulating and not on the procedures or functions. Data is hidden from functions and cannot access by external functions Program structure follows the "Bottom UP Approach." Example: C++, and C# (C sharp)

A non-structure oriented programming language is a type of programming language that does not require the use of structured programming concepts such as loops and conditionals. Such languages use a different approach to programming and are often less formalized in their syntax and grammar.

There is no specific structure rule for programming this language. Example: BASIC, COBOL, FORTRAN

14) What is modifier in C?

Modifiers specify the amount of memory needed to store a variable.

Modifiers are used to alter the storage capacity assigned to a variable by manipulating its data type, either by increasing or decreasing it.

The size of the data type used can affect the storage requirements of a variable in programming. For instance, the int data type on a 32-bit processor requires 4 bytes. However, using the long int data type which requires 8 bytes can increase the range available for a variable. On the other hand, using the short int data type which requires only 2 bytes can restrict the range of a variable.

15) What is the difference between variable declaration and variable definition in C?

The act of declaring a variable informs the compiler about the specific data type and size of the variable. On the other hand, when a variable is defined, the necessary memory is allocated for the variable.

It is possible to declare a variable multiple times in a program, but its definition can only occur once.

Variable declaration involves assigning properties and a name to a variable, while variable definition involves allocating storage space for the variable.

Cognizant OOPs Interview Questions

1) What are virtual Functions?

A virtual function is a type of function that can be overridden and inherited in object-oriented programming languages like C++ and Object Pascal. It enables dynamic dispatch, which is essential in the runtime polymorphism aspect of OOP.

2) What is overloading in OOPs?

Overloading refers to the concept of using the same method name for multiple functions, but with different sets of parameters. This technique is useful in reducing redundancy in code. During runtime, the actual method that is used is determined at compile time, which prevents errors from occurring.

3) What is overriding?

In object-oriented programming, overriding is a crucial feature that enables a subclass or child class to replace or modify an implementation of a function that has been provided by its super-class or parent class. This allows for specialized or customized implementations of functions within an inheritance hierarchy.

4) What is polymorphism?

Polymorphism is the ability for an object to have multiple types. This is often used when a parent class refers to an object from a child class. In Java, an object that can pass several IS-A tests is considered polymorphic.

5) What is data abstraction?

In object-oriented programming, abstraction is a fundamental principle that, alongside encapsulation and inheritance, forms the basis of designing effective and efficient code. By abstracting an object, a developer can reduce its complexity by hiding extraneous data and functionalities. This helps improve the application's performance as it can focus only on the relevant data, leading to better programming efficiency.

6) What is encapsulation?

Encapsulation is a fundamental principle of object-oriented programming that involves consolidating data and methods that manipulate that data within a single class. This concept is used to maintain the internal state of an object while safeguarding it from external manipulation. For instance, in Java, a class serves as a blueprint for encapsulating data.

7) What is Inheritance in java?

In programming, inheritance is a concept where new objects can acquire the attributes and functionalities of existing objects. The original objects are known as base or superclass while the newly created objects are referred to as derived or subclass. Inheritance allows for the reuse of code and simplifies the programming process.

8) What is the constructor?

A constructor is a special method that is used to set the initial state of a newly instantiated object. It is called immediately after the object is created in memory and can be used to assign values to the object's properties or set default values. In essence, constructors are used to prepare an object for use.

9) What is the destructor?

A destructor is a function that is automatically invoked during the deletion of a created object. It is crucial in freeing up memory space allocated to objects and removing files associated with them.
