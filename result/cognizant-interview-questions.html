Cognizant C Programming Interview Questions

1) What is a pointer? Define it.

A pointer is a variable that holds the address of another variable. It enables values of variables to be indirectly accessed, and their values to be manipulated.

2) What is a dangling pointer in C?

A dangling pointer is a type of pointer that does not point to a valid object of the appropriate type. It occurs when a pointer is present in the stack but not in the memory in a heap. For example, if a pointer is initialized with NULL and an attempt is made to deallocate space without first allocating it, a segmentation fault will occur.

A situation called dangling pointers can occur when an object is being deleted or deallocated, and the pointer data is not updated. This results in the pointer continuing to point to the memory location of the deallocated memory, even though the object is no longer accessible.

3) What is a memory leak in a heap?

This occurs in a heap when there exists a region of available memory, without any variable in the stack pointing towards it.

4) What is a data type?

A data type refers to the kind of values that can be assigned to a variable within a specific range. For example, "Int" is a data type that is capable of storing values between "-32768 to +32768" in the programming language of C.

5) What is the size of the integer data type?

The data type in question has a size of either 2 or 4 bytes.

6) What is malloc?

Malloc is a function used for dynamic memory allocation that allocates a requested size of bytes and returns the pointer to the first byte of the allocated space.

9) What is recursion?

Recursion involves a method calling itself continuously, either directly or indirectly. The function that allows for this process is also known as recursion.

10) What is the difference between a pre-increment operator and post-increment operator?

The pre-increment operator increases the variable value by one before it is assigned to the variable. In contrast, the post-increment operator increments the variable value by one after the value has been assigned to the variable.

12) What is embedded C?

Embedded C is a programming language that extends the capabilities of the C programming language.

The programming language Embedded C is utilized for the creation of applications based on microcontrollers.

Embedded C is a specialized variation of the C programming language that includes certain capabilities not found in standard C. For example, it offers fixed-point arithmetic, named address spaces, and the ability to address I/O hardware, which are essential for programming embedded systems.

Embedded systems such as cell phones and MP3 players use integrated C for programming and controlling their operations.

13) Which level is C language belonging?

C language is categorized as a middle-level language as it serves as a link between low-level machine languages and high-level languages.

C programming language is considered to be a middle-level language because it is more user-friendly compared to machine level languages, but its features are not as advanced as high-level languages offer.

14) What is the difference between structured oriented, object-oriented and non-structure oriented programming language?

A programming language that relies on a structured approach is known as a Structured Oriented Programming Language. This means that the code is broken down into organized blocks that can be easily understood and executed in a systematic manner.

In this type of language, large codes fragmented into small programs called functions. 14) What is modifier in C?

Modifiers are used to determine the amount of memory that should be reserved for a variable.

Modifiers are used in programming languages to alter the allocation of storage space for a variable by changing its basic data type. These changes can either increase or decrease the amount of storage space assigned.

One way to adjust the range of numerical data in a program is by selecting different data types with varying storage requirements. For instance, using the int data type on a 32-bit processor will use up 4 bytes of storage. If we need a wider range of values, we can opt for the long int data type which requires 8 bytes of storage. Conversely, if we want a narrower range, we can use the short int data type which uses only 2 bytes.

15) What is the difference between variable declaration and variable definition in C?

Variable declaration is a process that involves specifying the data type and size of a variable to the compiler. On the other hand, variable definition is the process of allocating memory to a variable.

It is possible to declare a variable multiple times in a program, but its definition can only occur once.

Variable declaration involves assigning properties and identification to a variable, while variable definition involves assigning storage space to a variable.

Cognizant OOPs Interview Questions

1) What are virtual Functions?

A virtual function or method is a feature found in object-oriented programming languages like C++ and Object Pascal. It is a function or method that can be inherited and overridden and allows for dynamic dispatch. This concept plays a crucial role in the polymorphism aspect of object-oriented programming at runtime.

2) What is overloading in OOPs?

Overloading is a technique utilized in programming to eliminate repetitive code by using the same method name with different parameters. This approach allows the program to determine the relevant method at compile time, removing the risk of encountering runtime errors.

3) What is overriding?

Overriding is a feature commonly used in object-oriented programming whereby a subclass or child class can implement its own version of a method that is already defined in its parent class or superclass.

4) What is polymorphism?

Polymorphism is a concept in object-oriented programming that enables an object to take on multiple forms or types. One common application of polymorphism is when a parent class refers to an object of a child class. In Java, an object that can pass more than one IS-A test is considered to be polymorphic.

5) What is data abstraction?

Abstraction is a crucial concept in object-oriented programming (OOP) that involves omitting unnecessary information about an object and retaining only what is relevant. This helps to simplify the system and improve its efficiency. Along with inheritance and encapsulation, it is one of the three fundamental principles of OOP.

6) What is encapsulation?

Encapsulation is an essential principle of object-oriented programming that involves integrating data within a class alongside the methods that operate on that data. In other words, encapsulation implies bundling data and methods into a single entity, such as a Java class. By doing so, the inner workings of an object can be kept hidden from external manipulation. Consequently, encapsulation safeguards an object's internal state from unauthorized access.

7) What is Inheritance in java?

Inheritance is a concept used in object-oriented programming where new objects can inherit properties from existing objects. It is achieved by creating a base class or superclass, which is a class that other classes can inherit from. Classes that inherit data and characteristics from a base class are known as subclasses or derived classes.

8) What is the constructor?

In object-oriented programming, a constructor is a special method that initializes a newly created object. It is called immediately after the object is allocated memory. Constructors can be used to set the initial values of an object, either to default values or specific values specified by the developer.

9) What is the destructor?

A destructor is a method that is invoked automatically during the destruction of an object. It is responsible for recovering the heap space occupied by the object and removing any associated files.

Cognizant Data structure Interview Questions

1) What is a Data Structure?

Data structure refers to the systematic arrangement and management of data in a way that enhances efficiency and ease of organization.

2) What have linked lists?

A linked list is a data structure where each node is connected to its next node in an ordered sequence. It is similar to an array, but more efficient since data can be inserted or deleted between existing nodes.

3) Is the array of data structure?

Certainly! The answer is affirmative, as the tool not only manages data but also organizes it in a systematic manner.

4) Define a binary search Tree?

A binary search tree, also called an ordered or sorted binary tree, arranges its nodes or keys in a sorted manner. In this kind of tree, the root node is greater than the left node and smaller than the right node.

5) What is a Binary Tree in the data structure?

A binary tree is a type of data structure characterized by two child nodes, one being on the left and the other on the right. In computer science, this type of tree is an evolved form of the linked list.

6) What is the dissimilarity between Push and Pop methods of the stack?

Pushing and popping are actions that pertain to how data is stored and retrieved from a stack. Pushing involves adding data to a stack, while popping involves fetching data from a stack, specifically the first piece of data in the stack.

7) What is the dissimilarity between a stack and the array?

A stack operates on the principle of LIFO (Last In First Out), meaning that data is accessed in a sequential order where the last data to be added is the first one to be removed. On the other hand, arrays do not have a specific order and can be accessed by referring to the indexed element within the array.

8) What are doubly linked Lists?

A doubly linked list is a type of linked list that enables traversal between data elements in both forward and backward directions. This is achieved through the use of two links in each node, one that connects to the next node and another that connects to the previous node.

9) What is a queue data structure?

A queue is a type of data structure that is linear in nature and operates on a fixed order. It follows the rule of First In First Out (FIFO), meaning the first item added to the queue will be the first one to get processed. One common example of a queue is the line of people waiting for customer service, where the first person to arrive will be the first one to be attended to. It is important to note that stacks and queues differ in terms of their deletion process.

10) What is a stack data structure?

The Stack is a data structure that comprises an organized list of elements of the same data type. It adheres to the principle of LIFO (Last In, First Out) or FILO (First In, Last Out). To add new elements, the push() method is used, while to remove elements, the pop() method is used.

11) What is a graph in the data structure?

A graph is a visual representation of a group of objects or nodes that are connected by links or connections. The nodes are represented by vertices, while the links that connect them are known as edges.

12) What is a priority queue?

A priority queue is a specialized kind of data type that includes a "priority" attribute attached to each element. While sharing similarities with queues and stacks, priority queues give priority to elements with higher priority values over those with lower values.

13) Show the time complexity of the linked list during insertion?

The efficiency of inserting or deleting elements in a singly linked list is determined by the location where it is performed. If the insertion or deletion happens at the end of the list, then it would require traversing the entire linked list, thus making the time complexity O(n).
