An m-order B-tree, also known as an M way tree, is a type of tree that has a specific property.

  1. Every terminal or outermost node only happens to be located at one specific level within a tree data structure.
  2. The topmost level of a tree data structure has a minimum of two child nodes and a maximum of m child nodes.
  3. Every non-leaf node in a B-tree must have a minimum of 'mdiv2' children (i.e., half of the maximum number of children 'm') and a maximum of 'm' children.
  4. Each node in a tree data structure contains a certain amount of data items. The number of data items present in a node is always one less than the number of children associated with that node.
