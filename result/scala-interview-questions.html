42) What is exception propagation in Scala?

Scala programming language enables the propagation of exceptions throughout the calling chain. In case an exception is raised in any function, it initially seeks a corresponding handler. If the handler is not present in that function, it forwards the exception to the calling method to look for a handler there. If a handler is found in the calling method, it catches the exception. However, if there is no handler found, it moves on to the next calling method in the chain. This entire process is referred to as exception propagation.
