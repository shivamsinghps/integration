Cloud Computing Interview Questions

The following are commonly asked Cloud Computing interview questions and answers in various companies, compiled for reference.

1) What is cloud computing?

Cloud computing is an innovative computer technology that operates over the internet. It leverages cloud resources to deliver services to users on demand, regardless of their location. It facilitates the ability to access numerous servers across the globe.

2) What are the benefits of cloud computing?

Cloud computing offers various advantages, which include:

  • Backing up and storing data refers to creating copies of digital information and keeping them in a secure location for future reference or in the event of data loss.
  • Powerful server capabilities.
  • Incremented productivity.
  • Highly efficient and economical in terms of both cost and time.
  • SaaS, which stands for Software as a Service, is a software licensing and delivery model where users access software applications over the internet on a subscription basis.

3) What is a cloud?

A cloud system is a comprehensive merging of various networks, hardware, storage, services, and interfaces designed to provide computing services to users. This system caters to three types of users:

  1. End users
  2. Business management users
  3. cloud service provider

4) What are the different data types used in cloud computing?

Cloud computing involves various types of data, such as emails, contracts, images, blogs, and more. With the increasing amount of data every day, new data types need to be created to accommodate this growth. For instance, storing a video would require a different data type than traditional forms of data.

5) Which are the different layers that define cloud architecture?

The cloud architecture is composed of various layers to enhance its performance and functionality. These layers are specifically designed to provide different services and meet different requirements.

  • A possible rephrased version could be: The CLC, which stands for Cloud Controller, is a component of cloud computing systems that manages the interaction between end users and the underlying infrastructure. Its main function is to provide a unified interface for provisioning, monitoring, and scaling resources across different cloud services. By acting as a gateway between the cloud and the user, the CLC simplifies the deployment and management of applications while ensuring optimal performance and utilization of resources.
  • Walrus
  • Cluster Controller
  • SC or Storage Controller is a crucial component in any computer system that manages the movement of data between computer storage and other components such as processors and input/output devices. Its main role is to ensure that the data is transmitted efficiently and reliably according to the command given by the processor.
  • Rephrase: The NC, also known as the Node Controller, is a component in a computer networking system that acts as a central point of control for data transfers and processing tasks within a cluster.

6) Which platforms are used for large scale cloud computing?

The utilization of the following platforms enables the management of extensive cloud computing operations:

  • Apache Hadoop
  • MapReduce

7) What are the different layers in cloud computing? Explain working of them.

The structure of cloud computing follows a hierarchy that consists of three layers.

IaaS (Infrastructure as a Service) is a cloud computing model where businesses can access and use virtual hardware resources such as processing power, memory, storage, and networking as a service over the internet.

PaaS, or "Platform as a Service," is a cloud computing solution that offers an application platform for software developers.

SaaS stands for "Software as a Service" and it allows users to access cloud-based applications without the need for installation on their systems. This means that the applications are hosted on the cloud and can be accessed directly by users.

8) What do you mean by software as a service?

SaaS is a significant component of cloud computing that offers cloud-based applications, as exemplified by the services provided by Google. With SaaS, users can store their files on the cloud and also create new documents.

9) What is the platform as a service?

The infrastructure layer in cloud architecture is responsible for providing resources like computers, storage, and network. On top of that, lies the platform layer, which facilitates complete virtualization of the infrastructure layer. This layer ensures that the infrastructure layer looks like a single server to users, making it invisible to the outside world.

10) What is on-demand functionality? How is it provided in cloud computing?

Cloud computing allows individuals to access virtualized IT resources as needed and is typically offered through shared pools of configurable resources, including networks, servers, storage, applications, and services.

11) What are the platforms used for large scale cloud computing?

Apache Hadoop and MapReduce are commonly utilized for accomplishing cloud computing on a large scale.

12) What are the different models for deployment in cloud computing?

Cloud computing has several deployment models that organizations can choose from. These models offer different levels of control, flexibility, and cost-effectiveness.

Private cloud

Public cloud

Hybrid cloud

Community cloud

13) What is private cloud?

Private clouds are utilized to ensure the confidentiality of significant operational data and other sensitive matters. These clouds offer a comprehensive infrastructure that can be entirely owned, managed and restricted to a particular organization or industry. In recent times, an increasing number of companies have opted for private clouds, primarily due to safety concerns. To ensure the safety of their data, some are gravitating towards virtual private clouds, which are operated by third-party hosting companies.

14) What is public cloud?

Public clouds are readily accessible to the general public and can be used and deployed by anyone. Examples of public clouds include Google and Amazon. Public clouds are primarily focused on providing services such as cloud applications, infrastructure, and platforms.

15) What are Hybrid clouds?

Hybrid clouds are a type of cloud computing architecture that combines the features of public and private clouds. This approach is becoming increasingly popular as it offers a more robust solution for cloud computing. By combining the strengths of both public and private clouds, organizations can create a customized cloud computing environment that delivers the flexibility of public clouds with the security and privacy of private clouds. Hybrid clouds enable organizations to have greater control over their data and applications, while also allowing them to benefit from the scalability and cost-effectiveness of public clouds.

16) What is the difference between cloud computing and mobile computing?

Mobile computing and cloud computing share similar principles. Cloud computing is the foundation for the concept of mobile computing, as it enables users to access the necessary data while on the go. On the other hand, mobile computing involves running applications on a distant server to allow users to access remote storage and management capabilities.

17) What is the difference between scalability and elasticity?

In the world of cloud computing, scalability and elasticity are both important characteristics that contribute to an architecture's flexibility in handling workload. Scalability enables an architecture to increase the amount of available resources in proportion to workload, while elasticity allows for dynamic commissioning and decommissioning of resources to meet changing demands. The effectiveness of elasticity is measured by how quickly resources are made available and how well they are utilized.

18) What are the security benefits of cloud computing?

Identity management in cloud computing involves granting access to application services. Therefore, cloud computing is utilized for managing identities.

The feature of permissions in cloud computing allows users to have control over the access of other users in the cloud environment. This ensures that the confidentiality and security of data stored in the cloud are maintained.

19) What is the usage of utility computing?

Utility computing is a software that enables organizations to select and manage their preferred cloud services. This system provides users with the ability to pay for only the services they use.

20) What is "EUCALYPTUS" in cloud computing? Why is it used?

EUCALYPTUS is a popular open-source software infrastructure used for implementing clusters in cloud computing platforms. It stands for Elastic Utility Computing Architecture For Linking Your Program To Useful Systems. With EUCALYPTUS, users have the ability to create public, private, and hybrid clouds and even their own data centers as private clouds. This allows organizations to efficiently utilize cloud computing resources and streamline their operations.

21) Explain System integrators in cloud computing.

A system integrator is responsible for devising a complex process for developing a cloud platform. Their expertise allows them to create more precise hybrid and private cloud networks, as they possess all the necessary knowledge for constructing data centers.

22) What are the open source cloud computing platform databases?

Open source cloud computing platform databases are becoming increasingly popular in the modern day. Examples of such databases include MongoDB, CouchDB, and LucidDB.

23) Give some example of large cloud provider and databases?

Google bigtable

Amazon simpleDB

Cloud based SQL

24) What is the difference between cloud and traditional datacenters?

The cost of implementing and maintaining a traditional datacenter is comparatively higher than that of cloud technology. Traditional databases often suffer from overheating issues and other hardware and software glitches which add to the maintenance cost.

25) What are the different in Software as a Service (SaaS)?

The Simple Multi-tenancy mode allows every user to have their own self-contained resources that are distinct from those of other users, resulting in an effective system.

In the fine grain multi-tenancy mode, multiple users are able to share resources while the system's functionality remains consistent.

26) Why API's is used in cloud services?

APIs (Application Programming Interfaces) are utilized in cloud platforms due to their ability to facilitate interactions between different applications and services.

The tool offers an alternative method where one is not required to write a comprehensive program.

Inter-application communication refers to the exchange of information between two or more applications to achieve a specific task or process.

The task of developing applications and connecting cloud services to other systems is performed by a software development process.

27) What are the advantages of cloud services?

The benefits of cloud services include:

  • Cost saving: It helps in the utilization of investment in the corporate sector. So, it is cost saving.
  • Scalable and Robust: It helps in the developing scalable and robust applications. Previously, the scaling took months, but now, scaling takes less time.
  • Time saving: It helps in saving time in terms of deployment and maintenance.

28) What are the different datacenters in cloud computing?

  1. Containerized datacenter
  2. Low density datacenter

29) What do you mean by CaaS?

CaaS refers to Communication as a Service, a term used in the telecom industry. It is a solution that provides enterprise users with a range of features such as desktop call control, unified messaging, and desktop faxing.

30) What do you mean by VPN? What does it contain?

A VPN is an acronym for Virtual Private Network which is a type of private cloud that ensures data security during cloud communication. It makes public networks behave like private networks.

31) What are the basic clouds in cloud computing?

Cloud computing encompasses three primary types of clouds.

  1. Professional cloud
  2. Personal cloud
  3. Performance cloud

32) What are the most essential things that must be followed before going for cloud computing platform?

  • Compliance
  • Loss of data
  • Data storage
  • Business continuity
  • Uptime
  • Ensuring data integrity is crucial when it comes to cloud computing. This involves making sure that the data stored in the cloud is accurate, consistent, and trustworthy. Any compromise in data integrity can lead to serious consequences for organizations using the cloud, including data loss and security breaches. Therefore, it is essential to establish proper measures and protocols for maintaining data integrity in the cloud.

37) How would you secure data for transport in cloud?

One concern that comes to mind is whether or not cloud data is secure. To ensure security, it's important to verify that data is not leaked during the transmission process between different points within the cloud. This can be done by implementing encryption keys with the data that is being transferred.
